## Receipt -- Generation & Tracking

### Overview

Automatic receipt generation for all financial transactions with sequential numbering, QR code verification, multiple output formats (PDF, email, print), void/reprint capabilities, and OCR upload for expense scanning.

### Capabilities by Role

- **Admin/Accountant**: Full access -- generate, void, reprint, view all receipts
- **Guardian**: View and reprint own receipts
- **Other roles**: No direct access

### Routes
