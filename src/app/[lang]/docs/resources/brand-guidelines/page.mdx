### Typography

#### Font Families

**Primary Font: Inter**
- Modern, clean, highly readable
- Excellent for UI and body text
- Variable font with 9 weights

**Arabic Font: Tajawal**
- Native Arabic design
- Matches Inter's x-height
- Supports RTL layouts

**Monospace: JetBrains Mono**
- For code and technical content
- Clear character distinction
- Ligature support

### Iconography

#### Icon Style
- **Style**: Outlined, 1.5px stroke
- **Corner Radius**: 2px
- **Grid**: 24x24px base
- **Padding**: 2px minimum
- **Library**: Lucide Icons

#### Icon Usage
- Functional icons: 16-24px
- Illustrative icons: 32-48px
- Always pair with labels for accessibility
- Maintain consistent stroke weight

## üìù Writing Guidelines

### Tone of Voice

#### Professional Yet Approachable
- ‚úÖ "Let's get your school set up in minutes"
- ‚ùå "Commence institutional configuration protocol"

#### Clear and Helpful
- ‚úÖ "Choose your subscription plan"
- ‚ùå "Select applicable service tier designation"

#### Encouraging
- ‚úÖ "Great job! Your students are all set"
- ‚ùå "Process completed successfully"

### Content Principles

#### 1. Clarity First
- Use simple, direct language
- Avoid jargon and technical terms
- Define acronyms on first use
- Provide context for complex concepts

#### 2. User-Centric
- Address user needs directly
- Use "you" and "your"
- Focus on benefits, not features
- Provide clear next steps

#### 3. Inclusive Language
- Gender-neutral pronouns
- Culturally sensitive examples
- Accessible to non-native speakers
- Respectful of all backgrounds

**For Parents:**
"Stay connected to your child's education journey"

**For Students:**
"Your academic life, organized and accessible"
