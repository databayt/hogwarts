## üöÄ Pro Tips

1. **Use Conductor for Complex Tasks**
   - Automatically coordinates multiple agents
   - Handles dependencies
   - Parallel execution when possible

2. **Chain Agents for Workflows**
   ```bash
   /agents/quality/test ‚Üí /agents/quality/perf ‚Üí /agents/quality/review-react
   ```

3. **Let Auto-Selection Work**
   - System picks best agent based on keywords
   - Override only when needed

4. **Leverage Parallel Execution**
   - Multiple agents can run simultaneously
   - Conductor handles coordination

5. **Use Appropriate Models**
   - Opus 4.1 for orchestration (quality)
   - Sonnet 4.5 for implementation (speed)
