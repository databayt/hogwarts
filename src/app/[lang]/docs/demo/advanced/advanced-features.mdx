**DO**:
- Use translation keys for all UI text

- Ignore font differences
- Forget to translate new features

### Best Practices for Auto-Marking

**DO**:
- Use auto-marking for objective, fact-based questions
- Provide clear, unambiguous answer options
- Include explanations for incorrect answers
- Review auto-graded results periodically for anomalies
- Set multiple accepted answers for fill-in-the-blank
- Use rubrics for manual grading consistency

**DON'T**:
- Rely solely on auto-marking (mix with subjective questions)
- Show correct answers immediately (reduces learning)
- Skip manual review of flagged questions
- Use vague or ambiguous question wording
- Penalize students for minor spelling variations
- Ignore student feedback about incorrect auto-grades

## Troubleshooting Advanced Features

### Multi-Language Issues

**Problem**: Text not translating
- Check translation key exists in dictionary
- Clear browser cache
- Verify language switcher cookie

**Problem**: RTL layout broken
- Check CSS dir attribute
- Verify Tailwind RTL utilities
- Test in different browsers

### Question Bank Issues

**Problem**: Questions not appearing in exam builder
- Verify question is published (not draft)
- Check subject/tag filters
- Ensure question has valid format

**Problem**: Auto-marking incorrect
- Review correct answer configuration
- Check for multiple correct answers
- Verify case sensitivity settings

### Analytics Issues

**Problem**: Dashboard not loading
- Check date range filters
- Reduce data scope (single grade/class)
- Clear cache and refresh

**Problem**: Export failing
- Check file size limits
- Try different export format
- Filter data to reduce size
