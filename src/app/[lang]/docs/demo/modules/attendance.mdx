## Troubleshooting

### Issue: Attendance not saving
**Possible Causes**:
- Network timeout (retry)
- Duplicate record (student already marked for this date)
- Invalid date (cannot mark attendance for future dates)

**Solution**:
- Check date selector (must be today or past)
- Refresh page and try again
- Check browser console for errors

---

### Issue: QR code not working
**Possible Causes**:
- QR code expired (>5 minutes old)
- Student not enrolled in selected class
- Student already checked in

**Solution**:
- Generate new QR code
- Verify student enrolled in class
- Check attendance list (student may be already present)

---

### Issue: Parent not receiving absence alerts
**Possible Causes**:
- Email incorrect in guardian profile
- Notifications disabled in settings
- Email in spam folder

**Solution**:
- Verify parent email in student profile
- Check Settings â†’ Notifications (ensure absence alerts enabled)
- Ask parent to check spam/junk folder
- Add `noreply@databayt.org` to contacts

---

### Issue: Cannot correct old attendance
**Possible Causes**:
- Correction window expired (>7 days for teachers)
- User lacks permission (must be admin)

**Solution**:
- Teachers: Request admin to make correction
- Admins: No restrictions, can correct anytime
