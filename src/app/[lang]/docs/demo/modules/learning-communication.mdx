**Steps**:

## Troubleshooting

### Common Issues & Solutions

#### **Library Module**:

**Issue**: Cannot borrow book - "Already borrowed maximum books"
- **Cause**: Student has reached borrowing limit (3 books for students)
- **Solution**: Return existing books first, or admin can increase limit

**Issue**: Book shows as "Available" but cannot borrow
- **Cause**: Book reserved for specific class/course
- **Solution**: Check if book is reserved, or contact librarian

**Issue**: Late fee calculation incorrect
- **Cause**: Due date or late fee rate misconfigured
- **Solution**: Admin checks Settings ‚Üí Library ‚Üí Late Fee Rate (default: SDG 2/day)

---

#### **LMS Module**:

**Issue**: Cannot access paid course after payment
- **Cause**: Payment not confirmed or enrollment failed
- **Solution**: Check Finance ‚Üí Invoices for payment status, or contact admin to manually enroll

**Issue**: Lesson video won't play
- **Cause**: Invalid video URL or format not supported
- **Solution**: Instructor checks video URL, ensure YouTube/Vimeo link is correct, or re-upload video

**Issue**: Progress not saving
- **Cause**: Browser cookies blocked or session expired
- **Solution**: Enable cookies, refresh page, log out and log back in

**Issue**: Certificate not generating after completion
- **Cause**: Not all lessons marked complete, or quiz requirements not met
- **Solution**: Check course progress, ensure 100% completion, complete any required quizzes

---

#### **Communication Module**:

**Issue**: Announcement not received by students
- **Cause**: Scope misconfigured or notification settings disabled
- **Solution**: Check announcement scope, verify student email settings, check notification preferences

**Issue**: Email notifications not sending
- **Cause**: Email server not configured or student email invalid
- **Solution**: Admin checks Settings ‚Üí Email ‚Üí SMTP Configuration, verify student emails in database

**Issue**: Cannot create announcement - "Permission denied"
- **Cause**: User role lacks permission to create announcements
- **Solution**: Teachers can only announce to their classes, other scopes require admin role

**Issue**: Scheduled announcement didn't send
- **Cause**: Server cron job not running or timezone misconfigured
- **Solution**: Admin checks server cron jobs, verify timezone in Settings ‚Üí General ‚Üí Timezone

## API Integration Examples

For developers integrating with these modules:

### Library API Endpoints

```typescript
// Get all books
GET /api/library/books
Query Params: ?genre=Science&status=available&limit=10

// Get book by ID
GET /api/library/books/:bookId

// Create borrowing
POST /api/library/borrowings
Body: { studentId, bookId, dueDate, notes }

// Return book
PUT /api/library/borrowings/:borrowingId/return
Body: { returnDate, condition, notes }

// Get student borrowing history
GET /api/library/borrowings/student/:studentId
```

### LMS API Endpoints

```typescript
// Get all courses
GET /api/lms/courses
Query Params: ?price=free&subject=science&limit=10

// Get course with content
GET /api/lms/courses/:courseId
Includes: chapters, lessons

// Enroll student
POST /api/lms/enrollments
Body: { studentId, courseId, paymentId? }

// Update lesson progress
PUT /api/lms/progress/:enrollmentId/lessons/:lessonId
Body: { completed: true, timeSpent: 600 }

// Get student progress
GET /api/lms/progress/student/:studentId/:courseId

// Issue certificate
POST /api/lms/certificates
Body: { enrollmentId }
```

### Announcements API Endpoints

```typescript
// Get all announcements (scoped by user)
GET /api/announcements
Query Params: ?scope=school&priority=important&unread=true

// Create announcement
POST /api/announcements
Body: {
  title, body, scope, scopeId?,
  priority, attachments?,
  scheduledAt?, notifyEmail, notifyApp, notifySms
}

// Mark as read
PUT /api/announcements/:announcementId/read

// Get engagement analytics
GET /api/announcements/:announcementId/analytics
```
