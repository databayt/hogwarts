### Data Privacy & FERPA Compliance

**Access Controls**:
- **Admin**: Full access to all students
- **Teachers**: Only students in their classes
- **Staff**: Limited access (library, clinic, etc.)
- **Guardians**: Only linked children
- **Students**: Own profile only

**Audit Trail**:
- Every profile view logged
- Every data change tracked (who, when, what changed)
- Audit logs retained for 7 years (compliance requirement)
- Export audit logs for specific student

**Data Export**:
- **Right to Access**: Guardians can request complete data export
- **Right to Deletion**: Soft delete (mark as deleted, retain for compliance)
- **Data Portability**: Export to standard formats (JSON, CSV, PDF)

**Sensitive Data**:
- Health records encrypted at rest
- Documents stored in secure cloud storage (Vercel Blob)
- Disciplinary records access-controlled
- Login credentials never displayed (hashed passwords)

## Troubleshooting

### Issue: Student not appearing in search
**Solution**:
- Clear filters (check status = ACTIVE)
- Try searching by student ID instead of name
- Refresh page

### Issue: Cannot add guardian
**Possible Causes**:
- Guardian email already exists (search instead of creating new)
- Invalid email format
- Missing required fields (name, relationship)

### Issue: Document upload fails
**Possible Causes**:
- File size > 10MB (compress or split)
- Unsupported format (use PDF, JPG, PNG only)
- Network timeout (try again)

### Issue: Status change blocked
**Possible Causes**:
- Outstanding fees (settle dues first for transfers)
- Active suspension (cannot graduate while suspended)
- Missing required information (reason, effective date)
