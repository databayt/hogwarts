## Agent Orchestration

### Parallel Execution

When tasks are independent, agents run in parallel:

```bash
/agents/orchestrate -p "In parallel:
1. Generate tests for StudentForm
2. Check security in auth module
3. Optimize Dashboard performance
4. Update documentation"
```

### Sequential Execution

When tasks have dependencies, agents run sequentially:

```bash
/agents/orchestrate -p "Sequential:
1. Create database migration
2. Update Prisma schema
3. Generate types
4. Create API endpoints"
```
