## Getting Started

### Basic Usage

```bash
# Use agents for specific expertise
/agents/nextjs -p "Help with routing"
/agents/react -p "Optimize component performance"
/agents/prisma -p "Fix N+1 queries"

# Use commands for quick tasks
/component StudentCard
/review
/security-scan
```

### Complex Features

For multi-step features, use the orchestrator:

```bash
/agents/orchestrate -p "Create student attendance tracking with:
- Calendar view
- Bulk actions
- Multi-tenant safety
- Arabic/English support
- Comprehensive tests"
```

## Platform-Specific Configuration

Claude Code is pre-configured for Hogwarts with:
- **Multi-tenant architecture** - All queries scoped by schoolId
- **Bilingual support** - Arabic (RTL) and English (LTR)
- **Mirror pattern** - Routes mirror component structure
- **Server-first** - Server components and actions
- **Type safety** - Strict TypeScript with no `any`
- **95%+ test coverage** - TDD approach
