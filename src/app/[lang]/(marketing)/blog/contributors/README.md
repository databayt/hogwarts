# Contributors Showcase

This directory contains introductions from all contributors to the Hogwarts project.

## How to Add Your Introduction

1. Fork the repository
2. Create a new file named `YOUR-GITHUB-USERNAME.mdx` in this directory
3. Use the following template:

```mdx
---
name: Your Name
github: your-github-username
date: YYYY-MM-DD
---

# Your Name

Hi! I'm contributing to Hogwarts because [your reason].

## What I'm Working On

- List your contributions
- Or areas of interest
- Or skills you bring

[Any additional content you want to share]
```

4. Submit a PR with title: `feat: add [Your Name] to contributors`

## Why We Do This

- It's a simple first contribution to get familiar with the codebase
- Helps build our community
- Shows that real people are behind this project
- Celebrates every contributor, no matter how small the contribution
