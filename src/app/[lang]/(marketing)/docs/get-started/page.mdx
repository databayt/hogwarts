## Localhost Setup

**01. Clone the repository**
```bash
git clone https://github.com/databayt/hogwarts.git && cd hogwarts
```

---

**02. Install dependencies**
```bash
pnpm install
```

---

**03. Create environment file**
```bash
cp .env.example .env
```

---

**04. Generate Prisma client**
```bash
pnpm prisma generate && pnpm prisma migrate dev
```

---

**05. Start dev server**
```bash
pnpm dev
```

---

**06. Open browser**

Go to http://localhost:3000

_For detailed setup, see [Localhost Setup](/docs/local-setup)._

## Submit a PR

**01. Create a new branch**
```bash
git checkout -b feature/your-name
```

---

**02. Stage and commit changes**
```bash
git add . && git commit -m "feat: add your change"
```

---

**03. Push to GitHub**
```bash
git push origin feature/your-name
```

---

**Checklist:**
- TypeScript passes (`pnpm tsc --noEmit`)
- Tests pass (`pnpm test`)
- Build succeeds (`pnpm build`)
- Database queries include `schoolId`

_See [Contributing Guide](/docs/contributing) for full details._

## First Contribution

Add your name to our contributors showcase:

**01. Create a new file**

Create `src/app/[lang]/(marketing)/blog/contributors/YOUR-NAME.mdx`

---

**02. Add your info**
```mdx
---
name: Your Name
github: your-github-username
date: 2024-11-21
---

# Your Name

Hi! I'm contributing to Hogwarts because [your reason].
```

---

**03. Submit PR**

Create PR with title: `feat: add [Your Name] to contributors`

**That's it! Welcome to Hogwarts.**
