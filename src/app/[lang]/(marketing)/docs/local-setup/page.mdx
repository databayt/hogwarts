## Prerequisites

**Node.js 18.17+**

Download from [nodejs.org](https://nodejs.org)

---

**pnpm 9.x**
```bash
npm install -g pnpm@9
```

---

**PostgreSQL Database**

- [Neon](https://neon.tech) - Free cloud PostgreSQL (recommended)
- [PostgreSQL](https://www.postgresql.org/download/) - Local install
- Docker:
```bash
docker run -d -p 5432:5432 -e POSTGRES_PASSWORD=postgres postgres:16
```

## Step 1: Clone Repository

**01. Clone the repository**
```bash
git clone https://github.com/databayt/hogwarts.git
```

---

**02. Navigate to project**
```bash
cd hogwarts
```

## Step 2: Install Dependencies

```bash
pnpm install
```

## Step 3: Environment Configuration

**01. Copy example file**
```bash
cp .env.example .env
```

---

**02. Configure .env file**

```bash
# Database Configuration
DATABASE_URL="postgresql://postgres:password@localhost:5432/hogwarts"
# For Neon (recommended): Sign up at neon.tech and paste your connection string

# NextAuth Configuration (required)
NEXTAUTH_URL="http://localhost:3000"
NEXTAUTH_SECRET="uyRBdRQWCbUC3Nk8EPR3g4DRBdRQWCbUC3Nk8EP"

# OAuth Providers (optional)
GOOGLE_CLIENT_ID=""
GOOGLE_CLIENT_SECRET=""
FACEBOOK_CLIENT_ID=""
FACEBOOK_CLIENT_SECRET=""

# Email (optional)
RESEND_API_KEY=""
EMAIL_FROM="noreply@hogwarts.edu"

# Monitoring (optional)
SENTRY_DSN=""
VERCEL_ANALYTICS_ID=""
```

## Step 4: Database Setup

**01. Generate Prisma client**
```bash
pnpm prisma generate
```

---

**02. Run migrations**
```bash
pnpm prisma migrate dev
```

---

**03. Seed database**
```bash
pnpm db:seed
```

## Step 5: Start Development Server

```bash
pnpm dev
```

**Success!** Hogwarts is running at http://localhost:3000
