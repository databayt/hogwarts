The mkan host onboarding has 3 stages, each opening with a ~4-second animated video on Airbnb's CDN. We replace the house-building animation with a school-building animation, keeping the same progressive 3-stage narrative (outline → details → complete).

## Goals

- Same progressive 3-stage narrative.
- Same flat / geometric animation style.
- ~4 seconds each.
- Transparent or gradient background preserved.

## Workflow

1. **Extract keyframes** — screenshot the peak frame from each original video.
2. **Edit keyframes** — Canva or Figma free tier: swap house for school silhouette (flag, clock tower, grid windows). Keep the flat illustration style and color palette.
3. **Animate** — Kling AI image-to-video with prompt `"smooth building animation, flat illustration style, minimal motion, 4 seconds"`.
4. **Background** — easiest is to bake in a gradient (mkan uses `from-orange-50 to-pink-50`; pick Hogwarts theme colors). Alternatives: CapCut (built-in BG removal), [Unscreen](https://www.unscreen.com) (watermark on free), [Remove.bg Video](https://www.remove.bg).
5. **Polish** — CapCut: trim to 4s, add seamless loop, export MP4 (H.264) for browsers, optionally WebM.

## Alternatives

**Lottie animations** — pixel-perfect, true transparency, tiny file size.

| Source                                             | Cost                | Skill                    |
| -------------------------------------------------- | ------------------- | ------------------------ |
| [LottieFiles](https://lottiefiles.com) marketplace | Free for many       | None — browse + download |
| [Rive](https://rive.app)                           | Free tier           | Low — visual editor      |
| After Effects + Bodymovin                          | Paid                | High — motion design     |
| Fiverr commission                                  | $50-100 for 3 clips | None                     |

**Keep originals** — Airbnb videos are abstract enough that most users won't connect them to Airbnb. Replace only static overview images with school-themed illustrations. Lowest effort, zero risk.

**CSS / SVG animations** — no external dependencies, perfect transparency, tiny size, fully customizable. Use Framer Motion (already in stack).

## Integration

Hogwarts onboarding has 15 steps. School-adapted videos slot into stage intros following mkan's pattern:

```tsx
const illustration = (
  <div className="mx-auto flex h-[300px] w-full max-w-xl items-center justify-center overflow-hidden rounded-xl sm:aspect-video sm:w-3/4 sm:rounded-2xl">
    <video
      ref={videoRef}
      className="h-full w-full object-cover"
      autoPlay
      muted
      playsInline
      preload="auto"
    >
      <source src="/videos/onboarding-stage-1.mp4" type="video/mp4" />
    </video>
  </div>
)
```

Store generated videos in `public/videos/`:

- `onboarding-stage-1.mp4` — school outline appears
- `onboarding-stage-2.mp4` — details added (windows, flag, playground)
- `onboarding-stage-3.mp4` — school completes
