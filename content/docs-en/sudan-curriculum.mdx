## 1. Overview

The Sudan national curriculum covers **grades 1-12** with a 6+3+3 structure (Elementary, Middle, Secondary). Content is extracted from official textbook PDFs published by the **National Center for Curricula and Educational Research (Bakht Al-Rida)**.

```
Sudan Education System (6+3+3)
================================
Elementary (1-6)  → المرحلة الابتدائية
Middle     (7-9)  → المرحلة المتوسطة
Secondary  (10-12) → المرحلة الثانوية
```

## 2. Slug Convention

Each curriculum uses a distinct slug pattern to prevent collision:

| Curriculum | Pattern             | Example      | Country | Curriculum Field |
| ---------- | ------------------- | ------------ | ------- | ---------------- |
| US K-12    | `us-g{N}-{subject}` | `us-g5-math` | `US`    | `us-k12`         |
| Sudan      | `sd-g{N}-{subject}` | `sd-g5-math` | `SD`    | `national`       |

**Key fields on `CatalogSubject`:**

```typescript
// US subject
{ slug: "us-g5-math", country: "US", curriculum: "us-k12", lang: "en", concept: "math" }

// Sudan subject
{ slug: "sd-g5-math", country: "SD", curriculum: "national", lang: "ar", concept: "math" }
```

## 5. Image Architecture

### Two Tiers of Thumbnails

US subjects have **per-subject thumbnails** uploaded individually from ClickView illustrations. Sudan subjects use **shared concept thumbnails** — 19 concept images serve all 97 subjects.

```
S3 Structure (grade-based concepts, shared across all curricula)
================================================================

  catalog/concepts/math-g5/thumbnail-sm.webp
  catalog/concepts/math-g5/thumbnail-md.webp
  catalog/concepts/math-g5/thumbnail-lg.webp
  catalog/concepts/math-g5/thumbnail-original.webp
  catalog/concepts/math-g5/banner-sm.webp
  catalog/concepts/math-g5/banner-md.webp
  catalog/concepts/math-g5/banner-lg.webp
  catalog/concepts/math-g5/banner-original.webp
```

Each key generates 4 WebP variants via Sharp: `sm` (200px), `md` (600px), `lg` (1200px), `original`.

### Image Resolution Chain

```
All subjects → getCatalogImageUrl(thumbnailKey, imageKey, size)
  1. thumbnailKey → catalog/concepts/math-g5/thumbnail-sm.webp            (shared concept S3)
  2. imageKey    → null                                                   (not set)
  3. concept     → SUBJECT_IMAGE_MAP["math"]                              (static fallback)
```

## 6. Multi-Curriculum Fallback

The `findCatalogSubjects()` function in `catalog-setup.ts` uses a 4-step progressive fallback:

```typescript
// Step 1: Exact match (country + curriculum)
//   SD school → country="SD", curriculum="national" → Sudan subjects

// Step 2: Broad match (country only)
//   SD school → country="SD" → any SD curriculum

// Step 3: Nearest match (same region/language)
//   Fallback to similar curricula

// Step 4: Universal fallback
//   US K-12 as the global default
```

When a school has `country: "SD"`, it automatically gets the Sudan curriculum. No configuration needed — the `inferCurriculum()` function maps `SD → "national"`.

### Stream Assignment (Grades 10-12)

Sudan secondary has academic streams. The `catalog-setup.ts` patterns handle this:

## 7. Data Pipeline

### Source → Database Flow

```
~/SD/*.pdf (260 raw files, ~112 unique textbooks)
    |
    ├── scripts/organize (copy + rename)
    |       → scripts/sudan-data/pdfs/grade-{N}/sd-g{N}-{subject}.pdf
    |
    ├── pdftotext (extract first 10-12 pages)
    |       → raw TOC text (Arabic chapter/lesson markers)
    |
    ├── parse TOC patterns (الوحدة, الباب, الدرس)
    |       → scripts/sudan-data/toc/sd-g{N}-{subject}.json (97 files)
    |
    ├── assemble-sudan-curriculum.ts
    |       → scripts/sudan-data/sudan-curriculum.json (master data file)
    |
    └── prisma/seeds/sudan-catalog.ts
            → CatalogSubject + CatalogChapter + CatalogLesson (upsert)
```

### TOC JSON Format

Each textbook produces one JSON file:

```json
{
  "subject": "math",
  "subjectNameAr": "الرياضيات",
  "grade": 3,
  "level": "elementary",
  "publisher": "المركز القومي للمناهج والبحث التربوي - بخت الرضا",
  "edition": "الطبعة الأولى 2021م",
  "chapters": [
    {
      "nameAr": "الأعداد ضمن 9999",
      "name": "Numbers within 9999",
      "slug": "numbers-within-9999",
      "sequenceOrder": 1,
      "lessons": [
        {
          "nameAr": "الأعداد حتى 999",
          "name": "Numbers up to 999",
          "slug": "numbers-up-to-999",
          "sequenceOrder": 1
        }
      ]
    }
  ]
}
```

### Master Data File

`scripts/sudan-data/sudan-curriculum.json` groups all TOC files by subject:

```json

      "name": "الرياضيات",
```

## 8. Seed Commands

```bash
# Seed Sudan curriculum (97 subjects, 406 chapters, 1049 lessons)
pnpm db:seed:single sudan-catalog

# Upload concept images to S3 and set thumbnailKey on all subjects
pnpm db:seed:single concept-images

# Rebuild master JSON from individual TOC files
npx tsx scripts/assemble-sudan-curriculum.ts
```

### Seed Registration

Both seeds are registered in `prisma/seeds/single.ts` with `global: true` (no schoolId required) and also called from the main `prisma/seeds/catalog.ts`.

### Adding a New Curriculum

To add a new country (e.g., Egypt `EG`):

1. Create TOC JSON files in `scripts/sudan-data/toc/` (or a new directory)
2. Add subject metadata to `scripts/assemble-sudan-curriculum.ts`
3. Add `EG → "national"` mapping in `inferCurriculum()` at `src/lib/catalog-setup.ts`
4. Add Arabic subject names to `NAME_TO_CONCEPT` in `prisma/seeds/concept-images.ts`
5. Run `pnpm db:seed:single {country}-catalog` and `pnpm db:seed:single concept-images`
