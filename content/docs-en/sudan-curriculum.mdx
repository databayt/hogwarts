Sudan covers grades 1-12 in a 6+3+3 structure (Elementary, Middle, Secondary). Content is extracted from textbook PDFs published by the **National Center for Curricula and Educational Research (Bakht Al-Rida)**. Subjects, chapters, and lessons live in the catalog with `country: "SD"` and `curriculum: "national"`.

## Slug convention

Each curriculum uses a distinct slug to prevent collision.

| Curriculum | Pattern             | Example      | `country` | `curriculum` |
| ---------- | ------------------- | ------------ | --------- | ------------ |
| US K-12    | `us-g{N}-{subject}` | `us-g5-math` | `US`      | `us-k12`     |
| Sudan      | `sd-g{N}-{subject}` | `sd-g5-math` | `SD`      | `national`   |

```ts
// US subject
{ slug: "us-g5-math", country: "US", curriculum: "us-k12", lang: "en", concept: "math" }

// Sudan subject
{ slug: "sd-g5-math", country: "SD", curriculum: "national", lang: "ar", concept: "math" }
```

## Image architecture

Sudan subjects share thumbnails via the [concept system](/docs/concept) — 23 concept images cover all 97 subjects. US subjects use per-subject thumbnails uploaded individually from ClickView.

```
catalog/concepts/math-g5/thumbnail-{sm,md,lg,original}.webp
catalog/concepts/math-g5/banner-{sm,md,lg,original}.webp
```

URL resolution falls through `thumbnailKey` → concept static fallback. The legacy `imageKey` field has been removed in favor of the unified `thumbnail`/`banner`/`cover`/`pdf` system.

## Data pipeline

```
~/SD/*.pdf (260 raw files, ~112 unique textbooks)
  → scripts/organize         → scripts/sudan-data/pdfs/grade-{N}/sd-g{N}-{subject}.pdf
  → pdftotext                → raw TOC text (Arabic chapter/lesson markers)
  → parse TOC patterns       → scripts/sudan-data/toc/sd-g{N}-{subject}.json (97 files)
  → assemble-sudan-curriculum.ts
                             → scripts/sudan-data/sudan-curriculum.json (master)
  → prisma/seeds/sudan-catalog.ts
                             → CatalogSubject + CatalogChapter + CatalogLesson (upsert)
```

### TOC JSON format

```json
{
  "subject": "math",
  "subjectNameAr": "الرياضيات",
  "grade": 3,
  "level": "elementary",
  "publisher": "المركز القومي للمناهج والبحث التربوي - بخت الرضا",
  "edition": "الطبعة الأولى 2021م",
  "chapters": [
    {
      "nameAr": "الأعداد ضمن 9999",
      "name": "Numbers within 9999",
      "slug": "numbers-within-9999",
      "sequenceOrder": 1,
      "lessons": [
        {
          "nameAr": "الأعداد حتى 999",
          "name": "Numbers up to 999",
          "slug": "numbers-up-to-999",
          "sequenceOrder": 1
        }
      ]
    }
  ]
}
```

The master file `sudan-curriculum.json` groups all TOC files by subject with `gradeChapters` indexed by grade number.

## Seed commands

```bash
pnpm db:seed:single sudan-catalog       # 97 subjects, 406 chapters, 1049 lessons
pnpm db:seed:single concepts      # Upload 19 concept images, set thumbnailKey
npx tsx scripts/assemble-sudan-curriculum.ts   # Rebuild master JSON from TOC files
```

Both seeds are registered with `global: true` in `prisma/seeds/single.ts` (no `schoolId` required).

## Adding a curriculum

For a new country (e.g. Egypt `EG`):

1. Add TOC JSON files under a per-country directory.
2. Add subject metadata to `scripts/assemble-sudan-curriculum.ts`.
3. Add `EG → "national"` mapping in `inferCurriculum()` at `src/components/catalog/setup.ts`.
4. Add Arabic subject names to `NAME_TO_CONCEPT` in `prisma/seeds/catalog/concepts.ts`.
5. Run `pnpm db:seed:single {country}-catalog` and `pnpm db:seed:single concepts`.
