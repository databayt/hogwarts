## MEA-specific platforms

Most relevant for our primary market (Sudan, Saudi Arabia, Egypt, South Africa, GCC).

1. [Classera](https://www.classera.com/) — Saudi-headquartered LMS + SIS hybrid. Strong in GCC private schools. Closed-source. Arabic-first UX.
2. [Almentor](https://www.almentor.net/) — Egyptian video-learning platform. Stronger on professional learning than K-12 ops, but creeping into school content distribution.
3. [Madrasati](https://madrasati.sa/) — Saudi Ministry of Education's national LMS. Government mandate, free for Saudi public schools. Not B2B for private market.
4. [SIS by Tatweer](https://tatweer.edu.sa/) — Saudi government student-information system. Public-sector default.
5. [Almis](https://www.almis-edu.com/) — Egyptian SIS focused on private schools. Closed-source, on-prem heritage.
6. [Edmodo](https://www.edmodo.com/) — Was widely deployed regionally before its 2022 shutdown; some MEA schools still asking for migrations.
7. [Eduvos](https://www.eduvos.com/) (formerly Pearson Africa) — South African higher-ed, but K-12 partner solutions exist via reseller channel.

## Global K-12 leaders

Mostly US / EU; relevant for feature benchmarking and any school migrating from these.

1. [PowerSchool](https://www.powerschool.com/) — $5.6B valuation, 60M+ students. Industry default in US public schools.
2. [Skyward](https://www.skyward.com/) — 2,500+ districts; Qmlativ platform.
3. [Infinite Campus](https://www.infinitecampus.com/) — 10 SEAs, 34+ US states.
4. [Classter](https://www.classter.com/) — 500+ institutions, 40+ countries. Closer to MEA than other globals.
5. [Blackbaud](https://www.blackbaud.com/) — 40+ years serving private K-12. Premium pricing.
6. [Veracross](https://www.veracross.com/) — Private K-12 specialist, strong for independent schools.
7. [FACTS SIS](https://factsmgt.com/) — Faith-based and private schools.
8. [Fedena](https://www.fedena.com/) — 22 core modules, global reach. Open-core model.
9. [OpenEduCat](https://openeducat.org/) — Open-source education management. Closest OSS analog to Hogwarts but India-focused and Odoo-based.

## vs Hogwarts

Where we differ from the field. This is positioning, not boast — every competitor wins on something.

| Dimension | Hogwarts | Most competitors |
| --------- | -------- | ---------------- |
| **License** | SSPL-1.0 (open source, self-hostable) | Closed source / proprietary |
| **Default locale** | Arabic-first, English parity | English-first; Arabic via translation layer |
| **RTL** | Logical CSS — works without overrides | Bolted on after the fact |
| **Multi-tenancy** | Subdomain-native, edge-resolved | Single-tenant or instance-per-school |
| **Custom domains** | Self-service via Upstash Redis lookup | Manual reverse-proxy or not supported |
| **Stack** | Next.js 16, React 19, Prisma 6, PostgreSQL on Neon | LAMP / Java / Rails legacy or proprietary |
| **AI in admission** | Anthropic-driven document classification + merit ranking + bank-receipt extraction | Manual review; OCR if any |
| **Payment gateways** | Stripe + bankak + tap + bank-transfer + cash + mobile-money | Stripe / PayPal only; rarely covers MEA payment methods |
| **Banking** | Plaid + Dwolla integration for school finance | External bookkeeping; no native banking |
| **Attendance modes** | 11 (manual, QR, barcode, kiosk, geofence, RFID, biometric, NFC, BLE, bulk-upload, face-recognition) | Manual + QR if lucky |
| **Messaging** | In-app real-time + email + push + WhatsApp Evolution API bridge | Email + maybe SMS |
| **Catalog model** | Shared global catalog with per-school overrides | Per-school content silos |
| **Curricula** | SD / US / IN / FR seeded; Sudan-specific bilingual subject names | One curriculum, US-default |
| **Architecture transparency** | Full architecture docs + per-feature CLAUDE.md guides | Opaque |
| **Contribution model** | Public GitHub, [shared-economy](/docs/shared-economy) revenue share | Closed development |

## Where competitors win

Honest accounting of what the field does better today.

- **PowerSchool / Skyward**: deeper US state-reporting integrations (FERPA-aligned exports, district aggregations).
- **Blackbaud / Veracross**: stronger fundraising / alumni / development modules for private schools.
- **Classter**: more mature multi-currency + multi-curriculum out of the box.
- **Almis**: long-standing relationships with Egyptian Ministry of Education for accreditation reporting.
- **Madrasati**: government mandate makes it the default for Saudi public schools at zero direct cost.
- **OpenEduCat**: deeper Odoo ecosystem integration if a school already runs Odoo.
