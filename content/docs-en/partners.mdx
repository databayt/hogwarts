This doc is the **non-obvious channel inventory**. Vendor credits live in [Sponsors](/docs/sponsors); cash grants in [Grants](/docs/grants); competitions in [Competitions](/docs/competitions). What's here is everything else — the partnerships that aren't a formal application but require relationship work.

_Last verified: 2026-05-18._

## Executive summary

We earn support by being useful to people whose mission overlaps ours. The audiences below sit outside the formal grant/accelerator/VC tracks. Each requires a different opening move: a ministry meeting starts with a memo, a CSR pitch starts with an existing employee referral, an NGO partnership starts with a co-delivery pilot.

We surface here several **less-obvious channels** the team often overlooks: Awqaf (religious endowments), Islamic Development Bank, diaspora bonds, Erasmus+ capacity building, Global Partnership for Education, EU Horizon, and corporate venture arms.

2. Pitch CSR as a marketing co-benefit (their logo on parent communications).
3. Tie the partnership to a measurable outcome (X schools, Y students reached, Z report-back).

## How to approach each audience

| Audience                | First move                                                | What to send first                                                                         |
| ----------------------- | --------------------------------------------------------- | ------------------------------------------------------------------------------------------ |
| Government ministry     | Letter from pilot principal + diplomatic-mission intro    | 2-page memo + demo URL + pilot reference call                                              |
| Corporate CSR           | Employee referral or warm intro                           | T6 (Outreach) + 1-page summary + impact metric proposal                                    |
| International NGO       | Co-delivery proposal, not grant ask                       | T4 (Outreach) + 2-page partnership memo                                                    |
| Bank                    | Integration pitch via existing partner (Bank of Khartoum) | T6 (Outreach) + integration spec                                                           |
| Telco                   | CSR + venture-arm dual pitch                              | T6 (Outreach) + 1-page sponsorship summary                                                 |
| Awqaf / IsDB            | Doctrinal-alignment memo + Sudanese-team intro            | Custom 2-page memo in Arabic + English; lead with Mudaraba/Shirkat al-A'mal/Waqf framework |
| Public figure (advisor) | Double-opt-in intro request                               | T8 (Outreach)                                                                              |
| Diaspora community      | Speaking at a community event + crowdfunding launch       | T19 (Outreach) + LaunchGood campaign                                                       |

## Anti-patterns

- **Mass-mailing a ministry mailbox.** It goes to a clerk. The minister does not have email.
- **Asking CSR for cash up-front.** Pitch the program; ask for the partnership; cash follows the relationship.
- **Treating NGOs like grant funders.** Most NGOs spend, they don't fund. Pitch co-delivery.
- **Using a public figure's name without permission.** Always double-opt-in.
- **Crowdfunding before you have a community.** A launched-and-failed campaign is worse than no campaign.
