A **concept** is a curriculum-agnostic identifier that maps every subject to shared visual assets. Arabic in Sudan and World Languages in the US share the same `languages` concept and the same S3 images.

```
Subject (any curriculum)
  └── concept: "math"
        ├── thumbnail: catalog/concepts/g{grade}-math/thumbnail-{size}.webp
        ├── banner:    catalog/concepts/g{grade}-math/banner-{size}.webp
        └── cover:     catalog/concepts/math/cover
```

## Cross-curriculum map

All 12 curricula — six deep (US, SD, GB, CAIE-IGCSE, IB-DP, CBSE) and six subjects-only nationals (SA, EG, AE, QA, KW, JO) — map their subjects to one of the 23 concepts. Sample mappings:

| Concept     | US                          | SD (Sudan)                          | GB                       | SA              |
| ----------- | --------------------------- | ----------------------------------- | ------------------------ | --------------- |
| `languages` | World Languages             | Arabic, Grammar, Literature, French | Modern Foreign Languages | Arabic          |
| `english`   | English Language Arts       | English                             | English                  | English         |
| `math`      | Math                        | Math                                | Mathematics              | Mathematics     |
| `science`   | Physical Science, Practices | Science, Engineering                | Science                  | Science         |
| `religion`  | Religion                    | Islamic Studies, Quran              | Religious Education      | Islamic Studies |
| `civics`    | Civics and Government       | Military Sciences                   | Citizenship              | Social Studies  |

For the full per-curriculum mapping (including Arabic subject names and grade ranges), see the per-national `prisma/seeds/catalog/{sa,eg,ae,qa,kw,jo}.ts` files and `scripts/sudan-data/sudan-curriculum.json`.

## S3 storage layout

Each concept has grade-specific S3 paths with 4 size variants. All 12 grades share the same source image per concept; grade-specific paths support future grade-specific imagery.

```
catalog/concepts/g1-math/thumbnail-sm.webp        (200px)
catalog/concepts/g1-math/thumbnail-md.webp        (600px)
catalog/concepts/g1-math/thumbnail-lg.webp        (1200px)
catalog/concepts/g1-math/thumbnail-original.webp
catalog/concepts/g1-math/banner-{sm,md,lg,original}.webp
catalog/concepts/math/cover
```

## Chapter rotation pools

Chapters within a subject rotate through a pool of 5 related concepts for visual variety. A math subject with 5 chapters gets thumbnails from math, science, computer-science, physics, economics — cycled by `sequenceOrder % poolSize`.

| Subject concept | Chapter rotation pool                                   |
| --------------- | ------------------------------------------------------- |
| `languages`     | languages, english, arts, history, geography            |
| `math`          | math, science, computer-science, physics, economics     |
| `english`       | english, languages, arts, history, sociology            |
| `science`       | science, biology, chemistry, physics, earth-science     |

## Seed pipeline

```bash
pnpm db:seed:single concepts     # 1. Process JPGs → S3 thumbnails (all 12 grades)
pnpm db:seed:single banners      # 2. Copy owned banners → concept paths
pnpm db:seed:single us           # 3. US subjects (sets concept + S3 keys)
pnpm db:seed:single sd           # 4. SD subjects (sets concept + S3 keys)
pnpm db:seed:single gb           # 5. GB deep tree (curriculum/uk)
pnpm db:seed:single cbse         # 6. CBSE deep tree (curriculum/in)
pnpm db:seed:single caie-igcse   # 7. Cambridge IGCSE deep tree (opt-in)
pnpm db:seed:single ib           # 8. IB-DP deep tree
pnpm db:seed:single sa           # 9. each national: sa/eg/ae/qa/kw/jo (or `full`)
```

The deep-tree sources (`gb`/`cbse`/`caie-igcse`/`ib`) all run through the shared
[tree engine](/docs/curriculum-engineering) and assign concepts the same way —
no per-curriculum banner upload needed.

## Adding a curriculum

To add a new curriculum that reuses concept images:

1. Define subjects with `concept` field pointing to one of the 23 concepts.
2. Set keys using the pattern:
   ```ts
   thumbnail: `catalog/concepts/g${grade}-${concept}/thumbnail`
   banner: `catalog/concepts/g${grade}-${concept}/banner`
   cover: `catalog/concepts/${concept}/cover`
   ```
3. Assign `color` from `CONCEPT_COLORS` or a custom hex.
4. Images are already on S3 — no upload required.
