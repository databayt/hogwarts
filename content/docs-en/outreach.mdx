This doc is the **outreach template library**. Copy the template that matches your audience, fill the `{{variables}}`, send. Short by policy — long cold emails get deleted.

_Last verified: 2026-05-18._

## How to use this doc

1. Pick the template that matches your audience (T1-T19 below).
2. Replace every `{{variable}}` with a specific value (no placeholders shipped).
3. Send via the correct channel (email, LinkedIn DM, WhatsApp).
4. Log the outreach in the relevant doc's status table ([Investors](/docs/investors), [Accelerators](/docs/accelerators), etc.).
5. Trigger follow-up at D+7 / D+14 / D+30 (T9-T12).

## Variable glossary

| Variable                         | Source                                                      |
| -------------------------------- | ----------------------------------------------------------- |
| `{{first_name}}`                 | The recipient's first name.                                 |
| `{{firm}}` / `{{org}}`           | Their organization name.                                    |
| `{{program}}`                    | Accelerator / grant / sponsor program name.                 |
| `{{traction_line}}`              | One sentence from [Traction](/docs/traction).               |
| `{{ask}}`                        | Specific ask — capital, intro, application, partnership.    |
| `{{deck_link}}`                  | Deck PDF or [Deck](/docs/deck) URL.                         |
| `{{data_room_link}}`             | Per-recipient gated data room link.                         |
| `{{demo_url}}`                   | Either `demo.databayt.org` or live pilot URL.               |
| `{{calendly}}`                   | Your scheduling link.                                       |
| `{{deadline_ref}}`               | Reference to the program's deadline if applicable.          |
| `{{alignment_angle}}`            | Which grant angle (A/B/C) matches the funder.               |
| `{{impact_metric}}`              | One quantified outcome.                                     |
| `{{open_source_angle}}`          | Why open-source matters for the program.                    |
| `{{integration_angle}}`          | Why the corporate-sponsor integration makes sense.          |
| `{{intro_giver}}`                | The mutual contact making a warm intro.                     |
| `{{target_name}}`                | Person being introduced to.                                 |
| `{{2_line_pitch}}`               | Your 2-line elevator pitch.                                 |
| `{{why_now}}`                    | One paragraph on why now is the right moment to intro.      |
| `{{thread_ref}}`                 | Subject line or 5-word reference to prior email.            |
| `{{new_traction_signal}}`        | A new data point since your last email.                     |
| `{{relevant_artifact}}`          | A link the recipient will care about (e.g., release notes). |
| `{{checkpoint_date}}`            | Future date to revisit (D+30 / D+90).                       |
| `{{milestone_to_check_back_on}}` | The milestone you'll have hit when you reply.               |
| `{{1_line}}`                     | Your one-line pitch (LinkedIn DM).                          |
| `{{conference}}`                 | Event you met at.                                           |
| `{{what_we_discussed}}`          | 1-sentence summary of the in-person conversation.           |
| `{{peer_school_count}}`          | Number of similar schools in the principal's network.       |
| `{{publication}}`                | Journalist's outlet.                                        |
| `{{angle}}`                      | The story angle you're pitching.                            |

## Cold outreach templates

### T1 — Cold investor email (high-signal)

**Audience**: VC partner, angel.
**Channel**: Email.
**Length**: 100-130 words.

```
Subject: Hogwarts — Arabic-first school OS, 1 paying pilot live

Hi {{first_name}},

{{firm}}'s portfolio in {{related_thesis}} caught my eye, especially {{recent_investment}} — we sit in adjacent territory.

I'm building Hogwarts: open-source, Arabic-first, multi-tenant K-12 school OS for MENA. We have one paying-pilot signed (King Fahad Schools, Sudan — live at [kingfahad.databayt.org](https://kingfahad.databayt.org)), a public demo, 4,912 source files, AR/EN parity, and AI integrations (Anthropic, OpenAI) wired into exam authoring and admission review.

We're raising $250K-$500K pre-seed bridge across SAFE + grants + accelerator stipend. {{ask}}

Deck: {{deck_link}}. 30 min to chat? {{calendly}}

Best,
{{your_name}}
```

**Channel**: Email.

```
{{your_name}}
```

### T7 — Warm-intro forwarding template

**Audience**: A mutual contact forwarding to a target investor / program.
**Channel**: Email (as the forwardable content).
**Length**: 70-90 words.

```
Hi {{intro_giver}},

Thanks for offering to introduce me to {{target_name}} at {{firm}}. Here's the forwardable for the intro:

---

Hi {{target_name}},

{{intro_giver}} suggested we connect. I run Hogwarts, an open-source Arabic-first school OS — one paying pilot live in Sudan, AI integrations, AR/EN parity.

We're raising a pre-seed bridge and {{firm}}'s focus on {{thesis}} maps closely. Quick 20-min intro call to see if there's fit?

Demo: {{demo_url}} · Deck: {{deck_link}}

Thanks,
{{your_name}}
```

### T8 — Double-opt-in intro request

**Audience**: Asking an advisor or pilot principal to intro you to a target.
**Channel**: Email.
**Length**: 80-100 words.

```
Subject: Intro request — {{target_name}} at {{firm}}

Hi {{advisor_name}},

I see {{target_name}} at {{firm}} is in your network. {{firm}} maps closely to Hogwarts's thesis ({{thesis}}) and {{why_now}}.

Would you be willing to make an intro? Happy to send you a forwardable that doesn't put you on the spot — you can edit anything before sending, or just send "thumbs up" and I'll write the intro to you instead.

If now isn't the right time, no worries at all — appreciate you considering it.

Thanks,
{{your_name}}
```

## Follow-up cadence

### T9 — Follow-up D+3 (light bump)

**Length**: 40-60 words.

```
Hi {{first_name}},

Light bump on {{thread_ref}} — wanted to make sure my note didn't get buried.

No pressure; if now isn't the right moment, happy to pick this back up later.

Best,
{{your_name}}
```

### T10 — Follow-up D+7 (bump with new signal)

**Length**: 50-70 words.

```
Hi {{first_name}},

Wanted to follow up on {{thread_ref}}.

Since I wrote: {{new_traction_signal}}. Wanted to flag in case it shifts the conversation.

Open to a 20-min chat whenever works for you.

Best,
{{your_name}}
```

### T11 — Follow-up D+14 (value-add)

**Length**: 60-80 words.

```
Hi {{first_name}},

Following up on {{thread_ref}}.

Thought you might find this relevant: {{relevant_artifact}} — covers {{topic}} which seemed adjacent to {{their_thesis_or_program}}.

Whether or not Hogwarts is a fit for you, happy to share more context in the space.

Best,
{{your_name}}
```

### T12 — Follow-up D+30 (closing the loop)

**Length**: 50-70 words.

```
Hi {{first_name}},

Closing the loop on {{thread_ref}}. I haven't heard back, so I'll assume the timing isn't right — totally understand.

Happy to reconnect in {{checkpoint_date}} or whenever you'd like.

Wishing you the best with {{their_initiative}}.

Best,
{{your_name}}
```

## Reply scripts

### T13 — Reply to rejection (graceful, door open)

**Length**: 50-70 words.

```
Hi {{first_name}},

Thanks for the clarity and the time. Totally understand.

If it'd be useful, I'd love to check back in {{checkpoint_date}} — by then we expect {{milestone_to_check_back_on}}, which might change the conversation.

In the meantime, if you ever come across someone in your network who'd find Hogwarts relevant, happy to be intro'd.

Thanks again,
{{your_name}}
```

### T14 — Reply to "send the deck"

**Length**: 30-50 words.

```
Hi {{first_name}},

Deck attached / linked: {{deck_link}}

For deeper diligence: data room at {{data_room_link}} (NDA-gated). Happy to set up a 30-min walkthrough at {{calendly}} or async in this thread.

Best,
{{your_name}}
```

### T15 — Reply to "not now"

**Length**: 60-80 words.

```
Hi {{first_name}},

Totally hear you on timing.

We expect to hit {{milestone_to_check_back_on}} by {{checkpoint_date}}. If that's a more interesting moment to chat, I'll plan to reach back out then — no need for you to track it.

If anything changes on your side in the meantime, you have my address.

Best,
{{your_name}}
```

## Anti-patterns

Things we never do in outreach:

- **No hype words** — "revolutionary," "best-in-class," "world-class," "disruptive." Quantify or don't claim.
- **No paragraphs longer than 4 lines** — cold emails get scanned, not read.
- **No two CTAs in one email** — pick one ask: meeting, intro, application, reply.
- **No "we'd love to connect"** — always offer something concrete (demo, deck, time slot).
- **No false urgency** — "limited spots," "closing soon" reads as desperate.
- **No copy-paste across recipients without changing variables** — recipients notice.
- **No replies to "no" with more pitching** — accept "no" gracefully (T13).
- **No following up more than 3 times** — silence after T12 is permission to stop.
- **No name-dropping without permission** — never use an advisor's name unless they've green-lit it.
- **No emojis in cold investor / grant emails** — fine in LinkedIn DMs to younger founders, never in formal outreach.

## Sending discipline

- **Send between 9-11 AM in recipient's timezone** when possible (most opens).
- **Avoid Mondays + Fridays** — Monday is busy, Friday gets buried.
- **One outreach batch per week** — Tuesday or Wednesday.
- **Log every send** in the relevant doc's status column.
- **Maintain a private response tracker** (Notion, sheet, or CRM) — separate from these public docs.
