Test the platform live. Open [demo.databayt.org/en/dashboard](https://demo.databayt.org/en/dashboard) and log in with `admin@databayt.org` / `1234`.

## Quick tour

Five places to explore in five minutes.

1. **Settings** — change role to see different dashboards.
2. **Appearance** — toggle dark / light mode and language (EN / AR).
3. **Students** — browse the list, add a student, view profile tabs.
4. **Timetable** — drag-and-drop schedule builder.
5. **Attendance** — mark daily attendance for a class.

## Demo accounts

All accounts use password `1234`.

| Email                     | Role       | Access                                  |
| ------------------------- | ---------- | --------------------------------------- |
| `admin@databayt.org`      | ADMIN      | Full access, manage everything          |
| `teacher@databayt.org`    | TEACHER    | Assigned classes, attendance, grades    |
| `student@databayt.org`    | STUDENT    | Personal schedule, grades, assignments  |
| `parent@databayt.org`     | GUARDIAN   | View linked children's data (read-only) |
| `staff@databayt.org`      | STAFF      | Limited admin on assigned entities      |
| `accountant@databayt.org` | ACCOUNTANT | Billing and invoices only               |

## Limitations

The demo resets daily and runs in sandbox mode:

- Email notifications disabled.
- File uploads limited to 1 MB.
- Cannot create new user accounts.
- School settings locked.
