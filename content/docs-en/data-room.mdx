This is the **index** of our data room. Contents are gated by NDA and shared per-recipient. The index itself stays public so anyone can see we have one.

_Last verified: 2026-05-18._

## What this doc is for

After the first call, serious counterparties ask "can you send the deck and a few backup docs?" This page is the reply menu — every artifact we can hand them, plus its readiness state. We don't ship until the relevant rows are Ready.

For the deck content, see [Deck](/docs/deck). For the numbers that fill it, see [Traction](/docs/traction).

## Access controls

- **Index** (this page): public.
- **Contents**: gated. Two-tier sharing:
  - **Tier 1** — non-sensitive (deck PDF, public demo creds, GitHub repo URL, references list): sent on request.
  - **Tier 2** — sensitive (financial model, pilot contracts, cap table numbers, security audit details): NDA required, watermarked per-recipient.
- **NDA template**: see TODO link below. Standard 1-year mutual NDA. Edit per-counterparty if needed.
- **Watermarking**: every Tier-2 PDF embeds `{{recipient_name}}` + `{{sent_date}}` in the footer.

## Refresh cadence

- **Monthly** (first Monday): captain reviews every TODO row and either upgrades to Ready or sets a target date.
- **Before each fundraising sprint**: re-verify Tier-2 contents; bump dates; refresh deck PDF.
- **Per-counterparty**: log who got which Tier in a private tracker (not this doc).

## Status legend

- **Ready** — exists, current, shareable.
- **Draft** — exists but needs polish or sanitization before sending.
- **TODO** — does not yet exist; create before claiming the data room is complete.
