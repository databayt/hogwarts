A school can enroll students, manage classes, track attendance, grade assignments, collect fees, and communicate with parents — all on a single platform with multi-tenant isolation. This page lists the epics that define MVP scope. For implementation status, file an issue or check existing ones on GitHub.

## Epic 01 — Infrastructure

Set up the core foundation where each school gets its own subdomain with isolated data. Read more in [Multi-Tenancy](/docs/multi-tenancy).

- Multi-tenant database schema with `schoolId` scoping.
- Subdomain routing (`school.databayt.org`).
- Next.js 16 + React 19 + Prisma 6 stack configured.
- CI/CD pipeline with automated tests.
- Sentry error monitoring.
- Internationalization (Arabic / English).
- Development, staging, and production environments.
- Database singleton pattern.
- Testing infrastructure (Vitest, Playwright).
- Multi-tenant isolation test suite.
- Edge middleware for subdomain rewriting.
- `getTenantContext()` helper.
- Type-safe environment configuration.
- Tailwind CSS 4 with RTL/LTR support.
- Layout + font configuration.

## Epic 02 — Authentication

Secure login with email, password, and social providers, plus role-based access for school staff. See [Authentication](/docs/authentication).

- NextAuth v5 with JWT strategy.
- Email/password with bcrypt.
- OAuth providers (Google, Facebook).
- Two-factor authentication (planned).
- All 8 roles: DEVELOPER, ADMIN, TEACHER, STUDENT, GUARDIAN, ACCOUNTANT, STAFF, USER.
- Route-level RBAC enforcement.
- Password reset flow.
- Session management with auto-renewal.
- User profile management.
- User invitation system (planned).
- Audit logging for sensitive operations (planned).

## Epic 03 — Configuration

Schools customize branding, academic structure, and settings during onboarding.

- School profile creation and management.
- Academic year setup with start/end dates.
- Term and period configuration.
- Year level / grade configuration.
- Department management.
- School branding (logo upload).
- Color picker for school theme.
- Timezone configuration.
- School settings page.
- Onboarding wizard.
- Logo preview and management.

## Epic 04 — Student

Manage the full student journey from enrollment to graduation with rich profiles and guardian links.

- Student enrollment form.
- Profile page with 8 tabs.
- Personal information.
- Academic information.
- Health records.
- Documents tab with upload.
- Achievements tracking.
- Disciplinary records.
- Fees tab with payment history.
- Search and advanced filtering.
- Guardian relationships and linking.
- Document upload and management.
- Photo upload and display.
- Bulk actions.
- Individual and bulk CSV export.
- Status transitions (enrolled / transferred / graduated).
- Previous education records.
- Emergency contacts.

## Epic 05 — Listings

CRUD for subjects, classes, teachers, parents, and staff with profile pages and tables.

- Subject catalog and CRUD.
- Class creation with capacity.
- Student enrollment in classes.
- Student-class conflict detection.
- Teacher assignment to classes.
- Teacher-subject allocation.
- Teacher profile creation, display.
- Qualifications management.
- Parent / guardian profile management.
- Staff profile creation.
- Department assignment.
- Class sections.
- Class roster export.
- Resource sharing within class.
- Class analytics dashboard.

## Epic 06 — Attendance

Daily attendance with simple marking or QR / location check-ins. See [Attendance](/docs/attendance).

- Daily attendance marking interface.
- Status options (present / absent / late / excused).
- Bulk marking.
- Reports by class and by student.
- Monthly summary.
- Parent absence notifications.
- Late arrival tracking.
- Attendance trend analytics.
- Data export.
- Teacher attendance tracking.
- Leave request integration.

## Epic 07 — Exam

Exam scheduling, grading, weighted result calculation, report card PDF. See [Exams](/docs/exams).

- Exam creation and scheduling.
- Type configuration (midterm / final / quiz).
- Grade entry interface.
- Grade scale configuration.
- Weighted-average result calculation.
- Report card generation.
- Report card templates.
- Result publication workflow.
- Parent result notifications.
- Class performance analytics.
- Subject-wise reports.
- Grade history tracking.
- Exam room allocation.
- Schedule conflict detection.
- Result export (PDF / CSV).

## Epic 08 — Timetable

Visual schedule builder with drag-and-drop, preventing double-booking of teachers or rooms. See [Timetable](/docs/timetable).

- Visual timetable builder.
- Drag-and-drop scheduling.
- Period-wise scheduling.
- Break management.
- Room allocation.
- Teacher conflict detection.
- Room conflict detection.
- Student conflict detection.
- Per-teacher, per-class, per-room views.
- Timetable export (PDF / print).

## Epic 09 — Admission

Public-facing application portal with multi-step form, document upload, and admin review. See [Admission](/docs/admission).

- Public application portal — no account required.
- 6-step multi-step application form.
- Auto-save with session resume.
- OTP-verified status tracking.
- Document upload.
- Self-service tour booking calendar.
- Tour booking management.
- Lead-capture inquiry form.
- Email notifications (confirmation, OTP).
- Campaign management (admin).
- Application review workflow.
- Admission decision (accept / reject / waitlist).
- Merit scoring.
- Stripe fee payment integration.
- Admission analytics.

## Epic 10 — Finance

Comprehensive finance module with double-entry bookkeeping, invoicing, payroll, and banking. Built beyond original MVP scope — 205 files across 40+ Prisma models. See [Fees](/docs/fees).

- Chart of accounts (double-entry bookkeeping).
- Fee structures and assignments.
- Payment processing and receipts.
- Invoicing with email delivery.
- Expense tracking with AI receipt extraction.
- Salary structures and management.
- Payroll runs with timesheets and slips.
- Banking with Plaid integration.
- Bank reconciliation.
- Budget management.
- Wallet system.
- Financial reports and dashboards.
- Scholarship management.
- Fines management.
- Role-based finance permissions.

## Epic 11 — Stream / LMS

School-scoped learning management system: courses, video player, enrollment, certificates. 102 files across 8 Prisma models.

- Admin course management (CRUD).
- Course categories.
- Video player with progress tracking.
- Student enrollment in courses.
- Course completion certificates.
- Rich-text editor for content.
- Admin analytics dashboard.
- Enrollment / completion email notifications.
- Course home page (hero, curriculum, skills).
- Sortable lesson ordering.
- Payment flow (cancel / success).
- Student dashboard with lesson progress.

## Epic 12 — Messaging

Real-time messaging with conversations, attachments, and search. 32 files across 12 Prisma models. See [Messages](/docs/messages).

- Chat interface with message bubbles.
- Conversation list and management.
- New conversation dialog.
- Attachment upload.
- Message search.
- Real-time updates via hooks.
- Multi-tenant authorization.
- Message grouping by date.

## Epic 13 — Catalog

Platform-wide educational content library sourced from ClickView. 9 Prisma models (global, no `schoolId` by design). See [Catalog](/docs/catalog).

- Catalog subjects (62 subjects).
- Catalog chapters (201 chapters).
- Catalog lessons (986 lessons).
- S3 / CloudFront image hosting.
- School-to-catalog bridge pattern.
- ClickView integration ([elementary](/docs/clickview-elementary), [middle](/docs/clickview-middle), [high](/docs/clickview-high)).

## What's not in scope

- Mobile native apps (mobile web works).
- Bus / transport routing.
- Inventory and asset tracking.
- Health clinic module.
- Library extensions beyond catalog.
