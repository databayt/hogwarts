The detailed walkthrough. For the fast path see [Get started](/docs/get-started).

## Prerequisites

- [Node.js](https://nodejs.org/) v22+ (LTS recommended)
- [Git](https://git-scm.com/)
- An editor — [VS Code](https://code.visualstudio.com/) recommended

## 1. Fork the repository

Open [github.com/databayt/hogwarts](https://github.com/databayt/hogwarts) and click **Fork** in the top-right.

<div className="my-8 overflow-hidden rounded-lg border">
  <Image
    src="/localhost/fork.png"
    alt="Fork button on GitHub"
    width={800}
    height={400}
    className="h-auto w-full"
  />
</div>

Keep default settings unless you need a custom name. Click **Create fork**.

<div className="my-8 overflow-hidden rounded-lg border">
  <Image
    src="/localhost/create-fork.png"
    alt="Create fork dialog"
    width={800}
    height={400}
    className="h-auto w-full"
  />
</div>

## 2. Clone in VS Code

Open VS Code → Git icon in the sidebar → **Clone**. Paste your fork URL or pick the repo from the GitHub picker.

<div className="my-8 overflow-hidden rounded-lg border">
  <Image
    src="/localhost/clone.png"
    alt="VS Code clone option"
    width={800}
    height={400}
    className="h-auto w-full"
  />
</div>

If you don't have the URL handy: open your fork on GitHub → click the green **Code** button → copy the HTTPS URL.

<div className="my-8 overflow-hidden rounded-lg border">
  <Image
    src="/localhost/clone-from-github.png"
    alt="Copy HTTPS URL from GitHub"
    width={800}
    height={400}
    className="h-auto w-full"
  />
</div>

Choose a parent directory (e.g. `~/Documents/projects/`).

<div className="my-8 overflow-hidden rounded-lg border">
  <Image
    src="/localhost/repo-destination.png"
    alt="Select destination directory"
    width={800}
    height={400}
    className="h-auto w-full"
  />
</div>

When the clone finishes, open the project in VS Code.

<div className="my-8 overflow-hidden rounded-lg border">
  <Image
    src="/localhost/open-repo.png"
    alt="Project opened in VS Code"
    width={800}
    height={400}
    className="h-auto w-full"
  />
</div>

## 3. Install, env, Prisma, dev

After cloning, follow [Get started](/docs/get-started) from step 3 onward.
