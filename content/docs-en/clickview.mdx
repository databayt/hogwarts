## Data shape

Each sub-topic in the inventory is keyed by slug and exposes:

```ts
type ClickViewTopic = {
  slug: string // kebab-case identifier
  name: string // display name
  videos: number // count of video resources
  resources: number // count of supporting resources
  coverId: string // ClickView cover image id
  subject: string
  topicGroup: string
}
```

To list every sub-topic for a tier:

```bash
jq '.tiers.elementary.subjects[] | .topics[]' \
  scripts/clickview-data/master-inventory.json
jq '.tiers.middle.subjects[] | .topics[]' \
  scripts/clickview-data/master-inventory.json
jq '.tiers.high.subjects[] | .topics[]' \
  scripts/clickview-data/master-inventory.json
```
