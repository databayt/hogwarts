_Last verified: 2026-05-18._

## Executive summary

Regional incubators give us things that global accelerators can't: hometown signal, on-the-ground mentor networks, local press, and partnerships with regional banks/ministries that already know us. Checks are smaller (€5K-€50K typically) but the local pull is asymmetrically valuable for a Sudanese-founded, MENA-operating EdTech.

Our position:

- **Sudanese-founded team** post-2023-conflict era — applies to Sudan and MENA reconstruction lanes.
- **King Fahad Schools pilot in Sudan** — already proves field-deployment in the target market.
- **Arabic-first product** — natural fit for AR-language-focused programs.
- **Open-source codebase** — increasingly accepted by MENA regional programs.

## Sudan context (anchor)

| Metric                                    | Value |
| ----------------------------------------- | ----- |
| Children out of school                    | 17M   |
| Days of learning lost                     | 500+  |
| Schools reopened (Khartoum, Feb 2026)     | ~35%  |
| UNESCO Transitional Education Plan budget | $580M |

This is the lane that the Sudan-focused programs care about most. We are the working OSS school OS that can be deployed at scale during reopening — and we already have a pilot live.

## Selection criteria

Apply if **at least two** of these are true:

1. The program is anchored in a country where we have an existing pilot, founder, or contributor.
2. The check or in-kind support compensates for the time cost of a 3-12 month program.
3. The mentor / network value is asymmetrically useful (regional bank, ministry, telco intros).
4. Equity ask is < 10% or zero.
5. The program does not require us to abandon Arabic-first or open-source positioning.

## Application workflow

For each program:

1. **Send a cold-program-manager email** (T2 or T3 AR-first in [Outreach](/docs/outreach)) BEFORE formal submission. Programs prefer warm starts.
2. **Adapt deck + 1-pager** to the program's filter — Orange Corners cares about Sudan-impact; Injini cares about EdTech-specific scaling.
3. **Confirm references** are reachable from the program's IP region.
4. **Submit formal application** via the program's portal.
5. **Follow-up cadence** — D+7, D+14 (T10, T11 in Outreach).
6. **Log status** in the table above and [Data Room](/docs/data-room).

## What we won't accept

- Programs that require us to abandon Arabic-first or open-source positioning.
- Pay-to-pitch / application-fee programs > $50.
- Programs that demand sector pivots (e.g., "drop EdTech, work on fintech").
- Programs whose only deliverable is "exposure" with no capital, mentorship, or network.
