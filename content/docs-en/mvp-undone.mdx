## What to DEFER (Not MVP)

These 57 features from the roadmap are **not essential** for min MVP:

- 2FA, audit logging, user invitation system
- Department management, branding/colors, timezone
- Student tabs: health, documents, achievements, disciplinary, fees
- Photo/document uploads, batch operations, status transitions
- Class capacity, conflict detection, qualifications, sections, analytics
- Attendance: student reports, monthly summary, notifications, exports, teacher tracking
- Exam: templates, publication workflow, parent notifications, analytics, room allocation
- Timetable: student conflict detection
- Admission: analytics/reports
- CI/CD, testing infrastructure
