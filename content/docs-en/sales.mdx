### 1. Work your warm network first

The fastest path to a pilot is someone who already trusts you. Before any cold prospecting, mine the relationships the team already has.

- [ ] Each team member lists every school, principal, or decision-maker they personally know → [Leads](/docs/leads#lead-sources)
- [ ] Enter each one in the [sales console](/sales) with the tier convention below — source Referral, `network` + `tier-a/b/c` tags → [Leads](/docs/leads#lead-sources)
- [ ] Complete missing fields per lead: full name, role, phone, WhatsApp, language preference (AR/EN), best channel, who can give a warm intro
- [ ] Research each school in the notes: curriculum, ~student count, operating mode (in-Sudan / displaced), loudest pain point, existing tools (Excel, WhatsApp groups, SMS)
- [ ] Reach out to Tier A contacts first using [T20 first-touch WhatsApp](/docs/outreach#t20-school-first-touch-whatsapp-sudan-tier-a-arabic-primary) and log every touch in the console → [Outreach](/docs/outreach)

**Tier convention** — tag every network lead with exactly one tier so the [sales console](/sales) Network filter splits cleanly:

| Tier | Meaning                                | Source    | Priority | Tags                                   |
| ---- | -------------------------------------- | --------- | -------- | -------------------------------------- |
| A    | Warm — you have the direct relay today | Referral  | Urgent   | `network`, `tier-a`, `<country>`       |
| B    | Warm but needs intro / more research   | Referral  | High     | `network`, `tier-b`, `<country>`       |
| C    | Cold — research before any outreach    | Cold Call | Medium   | `tier-c`, `<country>` _(no `network`)_ |

The current short list (11 schools, 2026-05-24) is seeded under `schoolId=platform` via `pnpm db:seed:single sales-network`. Re-run anytime — it skips rows that already exist by school name. Add new contacts directly in the console going forward.

## Reference

The condensed essentials. Full detail lives in the linked docs.

**Who we sell to** — K-12 schools in MEA, Arabic-first. Geography priority: Sudan (beachhead) → Saudi Arabia (largest market) → Egypt → wider GCC. Buyers are principals, school owners, education ministries, and multi-school networks. Full sizing in [Business model](/docs/business-model).

**Pricing**

| Tier       | Price               | Best for                         |
| ---------- | ------------------- | -------------------------------- |
| FREE       | $0                  | ≤ 100 students, evaluation       |
| PRO        | $1.50/student/month | Growing schools, custom branding |
| ENTERPRISE | $1.00/student/month | Networks, government contracts   |

A 500-student PRO school is $750/month; tier comparison and unit economics in [Business model](/docs/business-model).

**Top objections** (full scripts in [Leads](/docs/leads) and [Competitors](/docs/competitors))

- _"We already have a system"_ — Hogwarts is built for your region from day one: Arabic-first, multi-tenant isolation, MEA payment rails.
- _"It's too expensive"_ — a typical school saves ~40 admin hours/month; at $1.50/student it pays for itself in 2–3 months.
- _"We need to involve the ministry / board"_ — that's what the free 6-month pilot is for: measure, document, then decide.
