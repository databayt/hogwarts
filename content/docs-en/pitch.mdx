Hogwarts is an open-source school operating system. We automate the operational core of schools — admissions, classrooms, attendance, grading, fees, messaging — so educators spend less time on paperwork and more time on teaching.

For pricing and tiers see [Business model](/docs/business-model). For the commercial process see [Sales](/docs/sales).

## What we do

| Capability                    | Outcome                                                     |
| ----------------------------- | ----------------------------------------------------------- |
| Multi-tenant SaaS             | One platform serves many schools, isolated by `schoolId`    |
| Arabic-first UX with full RTL | Built for MEA, not bolted on                                |
| AI-assisted operations        | Document review, merit ranking, bank-receipt extraction     |
| Open source under SSPL-1.0    | Self-host free; managed SaaS for schools that prefer not to |

## Core capabilities

- **Academic** — student information, teacher and staff management, scheduling, attendance, grading, report cards.
- **Financial** — fee management, double-entry bookkeeping, invoicing, payroll, banking.
- **Communication** — parent portal, announcements, in-app messaging, WhatsApp Evolution API bridge.
- **Analytics** — role-specific dashboards, attendance trends, early-warning interventions, financial reports.

See [MVP](/docs/mvp) for the full epic-by-epic shipped state and [Roadmap](/docs/roadmap) for direction.

## Why open source

Student data is sensitive and curriculum changes constantly — schools deserve auditable, extensible code, and contributors deserve transparent attribution. SSPL-1.0 keeps the code free for self-hosters and protects against parasitic SaaS resale.

For the full open-source rationale see [Hogwarts overview](/docs/hogwarts).
