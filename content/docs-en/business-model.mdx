Tiered SaaS subscription with per-student pricing. Designed to scale from a 50-student private school to a national network of public schools. Live in production at [ed.databayt.org](https://ed.databayt.org); piloted with [King Fahad Schools](https://kingfahad.databayt.org) — the founding member of the [MENA-10 free pilot cohort](/docs/pilot). For commercial process see [Sales](/docs/sales); for the pilot motion see [Pilot](/docs/pilot); for the unified go-to-market see [GTM](/docs/gtm).

## Subscription tiers

| Tier           | Price               | Best for                       | Limits                                                                                                         |
| -------------- | ------------------- | ------------------------------ | -------------------------------------------------------------------------------------------------------------- |
| **FREE**       | $0                  | Small schools, evaluation      | 100 students, 1 GB storage, community support, core features                                                   |
| **PRO**        | $1.50/student/month | Growing schools                | 10 GB storage, priority support, custom branding, API 1K req/hr, advanced analytics, parent push notifications |
| **ENTERPRISE** | $1.00/student/month | Networks, government contracts | Unlimited storage and API, dedicated AM, 99.9% SLA, white-label, SSO (SAML/OAuth), custom integrations         |

All paid tiers include a **14-day free trial** with full feature access; no credit card required.

## Billing options

| Option     | Discount | Billing                             |
| ---------- | -------- | ----------------------------------- |
| Monthly    | —        | Per-student, metered monthly        |
| Annual     | 15% off  | Prepaid yearly commitment           |
| Multi-year | Custom   | Negotiated for government contracts |

**Payment methods**: Stripe Subscriptions (primary), bank transfer (enterprise / government), regional gateways (bankak, tap, mobile money) for MEA.

## Revenue streams

| Stream                    | Share | Description                                         |
| ------------------------- | ----- | --------------------------------------------------- |
| Per-student subscriptions | ~85%  | Recurring revenue from monthly / annual plans       |
| Enterprise contracts      | ~10%  | Multi-school + government deals with custom pricing |
| API overage               | small | Usage-based billing above tier limits               |
| White-label licensing     | small | Custom domain + complete rebranding fee             |
| Professional services     | small | Implementation, migration, training                 |
| Marketplace (future)      | —     | Commission on third-party integrations              |

## Growth strategy

| Phase                 | Plan                                                                                  |
| --------------------- | ------------------------------------------------------------------------------------- |
| 0. Free pilot cohort  | [MENA-10](/docs/pilot) — 10 free pilots, 6 months, feedback velocity → product polish |
| 1. Freemium           | FREE tier for ≤100 students; convert as they grow                                     |
| 2. Land + expand      | Pilot → 30%-off PRO Year 1, demonstrate ROI, propose network-wide ENTERPRISE          |
| 3. Government         | RFPs, custom compliance, multi-year volume contracts                                  |
| 4. Regional expansion | French / Spanish / Hindi / Mandarin locales; regional certifications                  |

See [GTM](/docs/gtm) for phase-gate logic — Phase 1 land-and-expand starts only after the cohort hits its conversion benchmarks. Free-pilot economics are intentional: ~$40 / pilot / month in infrastructure cost (Vercel + Neon + email), recovered if any single pilot converts at PRO 30%-off pricing.

## Cost structure

| Category      | Service                                                                       |
| ------------- | ----------------------------------------------------------------------------- |
| Hosting       | Vercel (serverless), Neon (PostgreSQL serverless)                             |
| Storage / CDN | AWS S3 + CloudFront (`cdn.databayt.org`)                                      |
| Email / SMS   | Resend, Twilio                                                                |
| Payments      | Stripe (2.9% + $0.30/transaction), regional gateways                          |
| Development   | Core team + community contributors via [shared economy](/docs/shared-economy) |
| Compliance    | Annual security audit, certifications                                         |

### Target margins

- **Gross margin**: 80%+ (cloud-native efficiency)
- **Operating margin**: 20%+ at scale
- **LTV:CAC ratio**: 10:1+

## Marketplace and partnerships

A future integration marketplace (70/30 partner/platform split) and reseller network are on the roadmap; not active today.

### Template store (future)

A planned add-on revenue stream: country-specific curriculum templates (Sudan curriculum, Saudi National Curriculum, IB MENA, Cambridge MENA, Azhar curriculum) sold to schools that adopt Hogwarts. Pricing range $5–$50 per template with subscription bundles. Activation trigger: ≥3 pilots ship with a custom curriculum we can productize. Not active in 2026 — see [GTM](/docs/gtm) Phase 3.

## Equity and revenue sharing

All employees receive equity (4-year vest, 1-year cliff). Open-source contributors earn through the [shared economy](/docs/shared-economy) framework.
