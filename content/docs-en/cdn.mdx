All platform UI assets are served from **S3 + CloudFront** via a single `asset()` helper function.

## Infrastructure

| Component        | Value                                                          |
| ---------------- | -------------------------------------------------------------- |
| S3 Bucket        | `hogwarts-databayt` (`us-east-1`)                              |
| CloudFront CNAME | `cdn.databayt.org` (preferred)                                 |
| CloudFront raw   | `d1dlwtcfl0db67.cloudfront.net` (still works — additive alias) |
| Cache            | `public, max-age=31536000, immutable` (1 year)                 |
| Env var          | `NEXT_PUBLIC_CDN_DOMAIN` (build-time, set on Vercel)           |

## Usage

```tsx
import { asset } from "@/lib/asset-url"

<Image src={asset("/icons/logo.png")} />
<Image src={asset("/illustrations/attendance.png")} />
<img src={asset("/icons/visa.svg")} />
<source src={asset("/media/story.mp4")} />
```

The `asset()` function prepends the CloudFront domain:

```
asset("/icons/logo.png")
→ https://cdn.databayt.org/icons/logo.png
```

## Naming Conventions

| Rule                     | Example            | Anti-pattern       |
| ------------------------ | ------------------ | ------------------ |
| kebab-case               | `data-science.png` | `dataScience.png`  |
| Lowercase                | `visa.svg`         | `Visa.svg`         |
| No spaces                | `hands-build.svg`  | `Hands Build.svg`  |
| No prefixes (default)    | `visa.svg`         | `payment-visa.svg` |
| Rename on collision only | `site-ball.png`    | Prefix everything  |

## Adding New Assets

1. Upload to the correct S3 directory (`icons/`, `illustrations/`, `photos/`, `animations/`, `media/`)
2. Use kebab-case, lowercase filenames
3. Set `Content-Type` and `CacheControl: "public, max-age=31536000, immutable"`
4. Reference with `asset("/category/filename.ext")`

Or use the migration script:

```bash
pnpm tsx scripts/migrate-assets-to-s3.ts          # dry run
pnpm tsx scripts/migrate-assets-to-s3.ts --upload  # upload
pnpm tsx scripts/migrate-assets-to-s3.ts --manifest # output JSON manifest
```
