# Icon System

Single-file SVG icon system using the shadcn/ui namespace pattern.

---

## Pattern

```tsx
// src/components/atom/icons.tsx
type IconProps = React.HTMLAttributes<SVGElement>

export const Icons = {
  logo: (props: IconProps) => (
    <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24" {...props}>
      <path fill="currentColor" d="..." />
    </svg>
  ),

  spinner: (props: IconProps) => (
    <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24" {...props}>
      <path fill="currentColor" d="..." />
    </svg>
  ),
}
```

---

## Usage

```tsx
import { Icons } from "@/components/atom/icons"

// Basic usage
<Icons.logo className="size-6" />

// With color inheritance
<div className="text-primary">
  <Icons.check /> {/* Inherits text-primary */}
</div>

// With animation
<Icons.spinner className="size-5 animate-spin" />
```

---

## File Location

**Single source of truth:** `src/components/atom/icons.tsx`

- 100+ icons already available
- Uses `currentColor` for theming
- Spreads `{...props}` for customization
- Standard viewBox: `0 0 24 24`

---

## SVG Requirements

| Property    | Value                    |
| ----------- | ------------------------ |
| viewBox     | `0 0 24 24` (UI icons)   |
| Colors      | `currentColor` only      |
| Props       | `{...props}` spread last |
| Fill/Stroke | `fill="currentColor"`    |
| xmlns       | Required                 |

## Icon Sub-Agent

Agent with Playwright MCP to fetch icons from Anthropic URLs.

### Workflow

1. Navigate to Anthropic URL (anthropic.com, claude.ai, docs.anthropic.com)
2. Extract SVG elements from the page
3. Convert to Icons namespace format
4. Append to `src/components/atom/icons.tsx`

### Invocation

```bash
# Fetch icon from Anthropic
/icon-fetch anthropic <icon-name>

# Generate from description
/icon-generate "a sparkle icon for AI features"
```

## Commands

| Command                 | Purpose                                 |
| ----------------------- | --------------------------------------- |
| `/icon-add <name>`      | Add icon from lucide to Icons namespace |

## Do's and Don'ts

**DO:**

```tsx
// ✅ Use Icons namespace
import { Icons } from "@/components/atom/icons"
<Icons.check className="size-5" />

// ✅ Use size-* utilities
<Icons.github className="size-4" />

// ✅ Use currentColor
<path fill="currentColor" d="..." />
```

**DON'T:**

```tsx
// ❌ Direct lucide import
import { Check } from "lucide-react"

// ❌ Hardcoded colors
<path fill="#000000" d="..." />

// ❌ Separate width/height
<Icons.github className="w-4 h-4" />
```
