## Trust artifacts

- **Security**: CodeQL workflow runs on every PR. Sentry monitors production. RBAC is `permissions.ts`-driven per feature (CASL). Row-level tenant isolation enforced on every server action.
- **Compliance posture**: GDPR / FERPA / COPPA / WCAG 2.1 AA documented in [PRD](/docs/prd).
- **License**: SSPL-1.0 — self-host free, commercial SaaS resale forbidden. Keeps code open without parasitic risk.
- **Open governance**: Contribution-tracked via [shared economy](/docs/shared-economy) — point-based valuation, Mudaraba/Shirkat al-A'mal/Waqf doctrinal layer.

## What is NOT traction (intellectual honesty)

The numbers above describe **build evidence**, not **revenue evidence**. We want investors and program managers to know the difference.

- **$0 MRR today.** The first pilot is free. Conversion is the gate.
- **One signed pilot is not a market.** It is a beachhead.
- **High line count, 52 tests.** We are not over-fit; we are not over-tested. Both will grow.
- **AI integrations are wired, not exotic.** Anthropic + OpenAI are common; the differentiator is Arabic-first product, not the model behind it.

We win the room by being honest about what is real and what is not.
