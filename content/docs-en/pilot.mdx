## Outreach Pipeline

| Day       | Action                                                  |
| --------- | ------------------------------------------------------- |
| Day 1     | Email + WhatsApp — introduce platform, offer FREE pilot |
| Day 4     | Follow-up with demo video link                          |
| Day 8     | Follow-up with different angle (time savings)           |
| Day 10-14 | Schedule 15-20 min live demo                            |
| Day 15    | Send pilot agreement                                    |

### Week-by-Week

| Week    | Actions                                                        |
| ------- | -------------------------------------------------------------- |
| Week 1  | Contact KAS (Egypt) + Westminster (Port Sudan) + WhatsApp SIGS |
| Week 2  | Call CITC + follow up Week 1 + try WES                         |
| Week 3  | Conduct demos for respondents                                  |
| Week 4  | Onboard first 1-2 pilot schools                                |
| Month 2 | Weekly check-ins, gather feedback                              |
| Month 3 | Convert to paid, collect testimonials                          |

## Onboarding Steps

| Phase        | Timeline   | Steps                                                   |
| ------------ | ---------- | ------------------------------------------------------- |
| Pre-Contact  | 2 days     | Research school, find decision maker                    |
| Outreach     | Days 1-8   | Email + WhatsApp, 2 follow-ups                          |
| Demo         | Days 10-15 | 15-20 min live demo, send pilot agreement               |
| Onboarding   | 1-2 days   | Create tenant, import data, admin + teacher training    |
| Active Pilot | Month 1-2  | Weekly check-ins, Month 1 review                        |
| Conversion   | Month 2-3  | Present results, propose paid plan, collect testimonial |

## Context

Sudan civil war (April 2023 — present) devastated education. Khartoum resumed its first school year in February 2026 after a 3-year hiatus.

| Stat                  | Value                                         |
| --------------------- | --------------------------------------------- |
| Children affected     | 17M                                           |
| Days of learning lost | 500+                                          |
| Children returned     | 3M                                            |
| Schools reopened      | ~35%                                          |
| Red Sea State schools | 600 reopened                                  |
| Gezira State          | All primary schools operational (Jun 2025)    |
| UNESCO plan           | Transitional Education Plan 2025-2027 ($580M) |
