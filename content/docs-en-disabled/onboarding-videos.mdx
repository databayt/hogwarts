# Onboarding Videos

Guide for adapting mkan's 3 Airbnb onboarding stage videos into school-themed animations for the Hogwarts onboarding flow.

## Adaptation Goal

Replace the house/home building animation with a school building animation while:

- Keeping the same progressive 3-stage narrative (outline, details, complete)
- Maintaining the same flat/geometric animation style
- Keeping videos at ~4 seconds each
- Preserving transparent or gradient backgrounds

## Recommended Free Workflow

### Step 1: Extract Keyframes

Use any screenshot tool to capture the peak frame from each original video at the moment showing the most complete structure.

### Step 2: Edit Keyframes

Use **Canva** (free tier) or **Figma** (free tier) to modify each keyframe:

- Swap the house shape for a school building silhouette
- Add school elements: flag, clock tower, windows in grid pattern
- Keep the same flat illustration style and color palette
- Maintain the same composition and framing

Alternative for generating keyframes from scratch: **Microsoft Designer** (Copilot Image Creator) - free, generates school buildings in matching flat style.

### Step 3: Animate

Use **Kling AI** free tier (Image-to-Video):

- Upload modified keyframe
- Prompt: "smooth building animation, flat illustration style, minimal motion, 4 seconds"
- Generate each stage separately

### Step 4: Handle Transparent Background

Options ranked by quality:

1. **Skip transparency** (recommended free approach) - Use a matching gradient background and bake it into the video. The mkan container already uses `from-orange-50 to-pink-50` gradient. Match Hogwarts' theme colors instead.

2. **CapCut** (free) - Has built-in background removal. Works for simple, high-contrast animations.

3. **Unscreen.com** - Free for short clips but may add watermark. Use free trial for 3 clips. URL: https://www.unscreen.com

4. **Remove.bg Video** - Limited free tier. URL: https://www.remove.bg

### Step 5: Final Polish

Use **CapCut** (free) to:

- Trim to exact 4-second duration
- Add loop point (videos should loop seamlessly)
- Export as MP4 (H.264) for browser compatibility
- Optionally export as WebM for better compression

## Alternative Approaches

### Approach A: Replace Videos with Lottie Animations

Instead of AI-generated videos, create **Lottie animations** for pixel-perfect results with true transparency.

| Tool                      | Cost                        | Skill Required             |
| ------------------------- | --------------------------- | -------------------------- |
| LottieFiles marketplace   | Free (many free animations) | None - browse and download |
| Rive                      | Free tier available         | Low - visual editor        |
| After Effects + Bodymovin | Paid                        | High - motion design       |
| Fiverr commission         | $50-100 for 3 clips         | None - hire someone        |

Search LottieFiles for "school building animation" or "building construction animation": https://lottiefiles.com

### Approach B: Keep Original Videos

The Airbnb videos are abstract enough that most users won't associate them with Airbnb. They show generic building animations, not branded content. Consider:

- Keep the 3 videos as-is
- Only replace the static overview images with school-themed illustrations
- Use Canva or Microsoft Designer to create school illustrations for free

This is the **lowest effort, zero risk** approach.

### Approach C: Use Static Illustrations Instead

Replace videos entirely with animated CSS/SVG illustrations:

- No external dependencies
- Perfect transparency
- Tiny file size
- Fully customizable
- Can be animated with Framer Motion (already in the stack)

## Integration in Hogwarts

The hogwarts onboarding currently has 15 steps. The school-adapted videos would slot into the stage intro pages following the same pattern as mkan:

```tsx
// Example: school onboarding stage intro
const illustration = (
  <div className="mx-auto flex h-[300px] w-full max-w-xl items-center justify-center overflow-hidden rounded-xl sm:aspect-video sm:w-3/4 sm:rounded-2xl">
    <video
      ref={videoRef}
      className="h-full w-full object-cover"
      autoPlay
      muted
      playsInline
      preload="auto"
    >
      <source src="/videos/onboarding-stage-1.mp4" type="video/mp4" />
    </video>
  </div>
)
```

Store generated videos in `public/videos/`:

- `onboarding-stage-1.mp4` - School outline appears
- `onboarding-stage-2.mp4` - Details added (windows, flag, playground)
- `onboarding-stage-3.mp4` - School building completes
