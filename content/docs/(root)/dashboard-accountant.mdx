The **Accountant Dashboard** provides financial officers with comprehensive tools for managing school finances, tracking collections, and monitoring budgets.

## Target Users

- **ACCOUNTANT**: School finance officers and accountants

## Layout Structure


## Financial Health Rings

3 financial metrics:

```tsx
const financialRings = [
  {
    label: "Collection",
    value: collectionRate,
    color: "#22c55e",
    current: collected,
    target: totalFees,
    unit: "collected",
  },
  {
    label: "Invoices",
    value: (paidInvoices / totalInvoices) * 100,
    color: "#3b82f6",
    current: paidInvoices,
    target: totalInvoices,
    unit: "paid",
  },
  {
    label: "Outstanding",
    value: 100 - (outstanding / totalFees) * 100,
    color: outstanding > 50000 ? "#ef4444" : "#f59e0b",
    current: outstanding,
    target: 0,
    unit: "pending",
  },
]
```

## Today's Transactions

Transaction list with status:

```tsx
{
  todaysTransactions.map((transaction) => (
    <div className="flex items-center justify-between rounded-lg border p-3">
      <div className="flex items-center gap-3">
        <div
          className={`rounded-full p-2 ${
            transaction.amount > 0 ? "bg-emerald-100" : "bg-red-100"
          }`}
        >
          {transaction.amount > 0 ? (
            <ArrowDownRight className="h-4 w-4 text-emerald-600" />
          ) : (
            <ArrowUpRight className="text-destructive h-4 w-4" />
          )}
        </div>
        <div>
          <p className="font-medium">{transaction.description}</p>
          <p className="text-muted-foreground text-sm">{transaction.time}</p>
        </div>
      </div>
      <div className="text-right">
        <p
          className={`font-bold ${transaction.amount > 0 ? "text-emerald-600" : "text-destructive"}`}
        >
          {transaction.amount > 0 ? "+" : ""}$
          {Math.abs(transaction.amount).toLocaleString()}
        </p>
        <Badge
          variant={transaction.status === "completed" ? "default" : "secondary"}
        >
          {transaction.status}
        </Badge>
      </div>
    </div>
  ))
}
```

## Financial Calendar

Upcoming financial events:

```tsx
{
  financialCalendar.map((event) => {
    const daysUntil = Math.ceil(
      (eventDate.getTime() - Date.now()) / (1000 * 60 * 60 * 24)
    )

    return (
      <div className="flex items-center justify-between rounded-lg border p-3">
        <div className="flex items-center gap-3">
          <div
            className={`rounded-full p-2 ${
              event.priority === "critical"
                ? "bg-red-100"
                : event.priority === "high"
                  ? "bg-amber-100"
                  : "bg-blue-100"
            }`}
          >
            <Calendar className="h-4 w-4" />
          </div>
          <div>
            <p className="font-medium">{event.event}</p>
            <p className="text-muted-foreground text-sm capitalize">
              {event.type}
            </p>
          </div>
        </div>
        <div className="text-right">
          <Badge
            variant={
              event.priority === "critical"
                ? "destructive"
                : event.priority === "high"
                  ? "default"
                  : "secondary"
            }
          >
            {daysUntil <= 0 ? "Today" : `${daysUntil} days`}
          </Badge>
          <p className="text-muted-foreground mt-1 text-xs">
            {format(eventDate, "MMM d, yyyy")}
          </p>
        </div>
      </div>
    )
  })
}
```

Event types: reporting, audit, tax, budget
