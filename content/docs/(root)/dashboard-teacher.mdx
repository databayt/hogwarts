The **Teacher Dashboard** provides educators with tools to manage their daily teaching activities, track student progress, and handle administrative tasks efficiently.

## Target Users

- **TEACHER**: Classroom teachers and instructors

## Layout Structure

```tsx
<div className="space-y-6">
  {/* Welcome Banner */}
  <WelcomeBanner
    userName={user.name}
    role="Teacher"
    subtitle={`You have ${todaysClasses.length} classes scheduled for today`}
  />

  {/* Key Metrics Row - 4 columns */}
  <div className="grid gap-4 grid-cols-2 lg:grid-cols-4">
    <MetricCard title="Today's Classes" ... />
    <MetricCard title="Total Students" ... />
    <MetricCard title="Pending Grading" ... />
    <MetricCard title="Attendance Due" ... />

  <QuickActions actions={...} locale={locale} />

  {/* Main Content Grid - 3 columns */}
  <div className="grid gap-6 lg:grid-cols-3">
```

## Key Metrics

**4-column responsive grid**:

| Metric | Icon | Color Logic |
|--------|------|-------------|
| Today's Classes | BookOpen | Blue |
| Total Students | Users | Purple |
| Pending Grading | FileText | Red if >5, else orange |
| Attendance Due | CheckCircle | Amber if >0, else green |

```tsx
<MetricCard
  title="Pending Grading"

  href={`/${locale}/s/${school?.domain}/assignments`}

```

## Today's Schedule

2-column span card with `ScheduleItem` components:

```tsx
{data.todaysClasses.map((cls, index) => (
  <ScheduleItem
    key={cls.id}
    time={cls.time}
    title={cls.name}
    subtitle={`Room ${cls.room} • ${cls.students} students`}
    badge={index === 0 ? "Next" : undefined}
    isActive={index === 0}
  />
))}
```

Features:
- Time display
- Class name
- Room and student count
- "Next" badge for upcoming class
- Active state highlighting

## Pending Assignments

List with due date indicators:

```tsx
{pendingAssignments.map((assignment) => {
  const dueDate = new Date(assignment.dueDate);
  const isOverdue = dueDate < new Date();
  const isDueToday = isToday(dueDate);
  const isDueTomorrow = isTomorrow(dueDate);

  return (
    <div className="flex items-center justify-between p-3 rounded-lg border">
      <div>
        <p className="font-medium">{assignment.title}</p>
        <p className="text-sm text-muted-foreground">
          {assignment.className} • {assignment.submissionsCount} submissions
        </p>
      </div>
      <Badge variant={isOverdue ? "destructive" : isDueToday ? "default" : "outline"}>
        {isOverdue ? "Overdue" : isDueToday ? "Due Today" : format(dueDate, "MMM d")}
      </Badge>
    </div>
  );
})}
```

## Class Performance Summary

Performance by class with status badges:

| Grade Range | Badge Style |
|-------------|-------------|
| >= 80% | Default (emerald) |
| >= 60% | Secondary |
| < 60% | Destructive |

```tsx
<Badge
  variant={cls.average >= 80 ? "default" : cls.average >= 60 ? "secondary" : "destructive"}
>
  {cls.average >= 80 ? "Excellent" : cls.average >= 60 ? "Good" : "Needs Attention"}
</Badge>
```

## Upcoming Deadlines

Task list with countdown:

```tsx
const daysLeft = Math.ceil((dueDate.getTime() - Date.now()) / (1000 * 60 * 60 * 24));

<Badge variant={daysLeft <= 2 ? "destructive" : daysLeft <= 7 ? "secondary" : "outline"}>
  {daysLeft <= 0 ? "Today" : daysLeft === 1 ? "1 day" : `${daysLeft} days`}
</Badge>
```

## Error Handling

Comprehensive try-catch with user feedback:

```tsx
try {
  data = await getTeacherDashboardData();
} catch (error) {
  console.error("[TeacherDashboard] Error:", error);
  return (
    <Card>
      <CardContent className="p-6">
        <h3>Unable to Load Dashboard</h3>
        <p>Please try refreshing the page.</p>
      </CardContent>
    </Card>
  );
}
```
