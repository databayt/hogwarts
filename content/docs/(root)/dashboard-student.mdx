The **Student Dashboard** provides students with a personalized view of their academic progress, upcoming assignments, recent grades, and daily schedule.

## Target Users

- **STUDENT**: Enrolled students

## Layout Structure

```tsx
<div className="space-y-6">
  {/* Welcome Banner */}
  <WelcomeBanner
    userName={user.name}
    role="Student"
    subtitle={assignments.length > 0 ? `${assignments.length} assignments due soon` : "You're all caught up!"}
  />

  {/* Key Metrics Row - 4 columns */}
  <div className="grid gap-4 grid-cols-2 lg:grid-cols-4">
    <MetricCard title="Attendance" ... />
    <MetricCard title="Average Grade" ... />
    <MetricCard title="Assignments Due" ... />
    <MetricCard title="Classes Today" ... />
  </div>

  {/* Attendance Progress Card */}
  <AttendanceGoalCard />

  {/* Quick Actions */}
  <QuickActions actions={...} locale={locale} />

  {/* Main Content Grid - 3 columns */}
  <div className="grid gap-6 lg:grid-cols-3">
    <TodaysScheduleCard className="lg:col-span-2" />
    <ActivityRings activities={activityData} />
  </div>

  {/* Secondary Content Grid - 2 columns */}
  <div className="grid gap-6 md:grid-cols-2">
    <PerformanceGauge />
    <UpcomingAssignmentsCard />
    <RecentGradesCard />
    <AnnouncementsCard />
  </div>

  {/* Grade Trend Chart */}
  <ComparisonLineChart ... />

  {/* Academic Summary Stats */}
  <AcademicSummaryCard />

  {/* Progress Cards - 3 columns */}
  <div className="grid gap-6 md:grid-cols-3">
    <ProgressCard title="Attendance Goal" ... />
    <ProgressCard title="Assignments Completed" ... />
    <ProgressCard title="Term Progress" ... />
  </div>

```

## Welcome Banner

Dynamic subtitle based on assignment count:

```tsx
<WelcomeBanner
  userName={user.name || user.email?.split("@")[0]}
  role="Student"
  subtitle={
    data.upcomingAssignments.length > 0
      ? `You have ${data.upcomingAssignments.length} assignments due soon`
      : "You're all caught up!"
  }
/>
```

## Key Metrics

| Metric | Icon | Color Logic |
|--------|------|-------------|
| Attendance | Calendar | Green if >=85%, else amber |
| Average Grade | GraduationCap | Green >=80%, blue >=60%, else amber |
| Assignments Due | FileText | Red if >3, else purple |
| Classes Today | BookOpen | Blue |

```tsx
<MetricCard
  title="Average Grade"
  value={averageGrade > 0 ? `${averageGrade.toFixed(0)}%` : "N/A"}
  iconName="GraduationCap"
  iconColor={averageGrade >= 80 ? "text-green-500" : averageGrade >= 60 ? "text-blue-500" : "text-amber-500"}
/>
```

## Activity Rings

3 student progress metrics:

```tsx
const activityData = [
  {
    label: "Attendance",
    value: attendanceSummary.percentage,
    color: percentage >= 85 ? "#22c55e" : "#f59e0b",
    current: presentDays,
    target: totalDays,
    unit: "days",
  },
  {
    label: "Grades",
    value: averageGrade,
    color: averageGrade >= 80 ? "#22c55e" : averageGrade >= 60 ? "#3b82f6" : "#ef4444",
    current: Math.round(averageGrade),
    target: 100,
    unit: "%",
  },
  {
    label: "Tasks",
    value: 100 - (assignments.length * 20),
    color: "#8b5cf6",
    current: assignments.length,
    target: 5,
    unit: "due",
  },
];
```

## Upcoming Assignments

List with status badges:

```tsx
{upcomingAssignments.map((assignment) => {
  const dueDate = new Date(assignment.dueDate);
  const daysUntil = differenceInDays(dueDate, new Date());
  const isOverdue = daysUntil < 0;

  return (
    <div className="flex items-center justify-between p-3 rounded-lg border">
      <div>
        <p className="font-medium">{assignment.title}</p>
        <p className="text-sm text-muted-foreground">
          {assignment.subject} • {assignment.className}
        </p>
      </div>
      <Badge variant={
        isOverdue || assignment.status === "NOT_SUBMITTED" ? "destructive" :
        isToday(dueDate) ? "default" :
        isTomorrow(dueDate) ? "secondary" : "outline"
      }>
        {isOverdue ? "Overdue" : daysUntil === 0 ? "Due Today" : `${daysUntil} days`}
      </Badge>
    </div>
  );
})}
```

## Recent Grades

Grade cards with score and percentage:

```tsx
{recentGrades.map((grade) => (
  <div className="flex items-center justify-between p-3 rounded-lg border">
    <div>
      <p className="font-medium">{grade.examTitle}</p>
      <p className="text-sm text-muted-foreground">{grade.subject}</p>
    </div>
    <div className="text-right">
      <p className="font-bold">{grade.marksObtained}/{grade.totalMarks}</p>
      <Badge className={grade.percentage >= 80 ? "bg-emerald-100 text-emerald-700" : ""}>
        {grade.percentage.toFixed(0)}%{grade.grade ? ` • ${grade.grade}` : ""}
      </Badge>
    </div>
  </div>
))}
```


## Error Handling

```tsx
try {
  data = await getStudentDashboardData();
} catch (error) {
  console.error("[StudentDashboard] Error:", error);
  return <ErrorCard message="Unable to load dashboard" />;
}
```
