## Phase 4: Quality & Production

<div className="text-sm text-muted-foreground mb-4">Week 5-6</div>

### P4.1 Test Coverage Expansion

<div className="p-4 border rounded-lg my-4">
  <div className="flex justify-between items-center">
    <span>Current</span>
    <span className="font-bold">15 test files</span>
  </div>
  <div className="flex justify-between items-center">
    <span>Target</span>
    <span className="font-bold text-green-600">50 test files</span>
  </div>
</div>

**Priority Areas:**
1. Server Actions: `students`, `attendance`, `settings`, `classes`, `teachers`
2. Multi-tenant isolation tests
3. Form validation schemas
4. Guardian linking workflow

**Skip for MVP:** UI component tests, snapshot tests, E2E expansion

---

### P4.2 Mobile Responsive Audit

**Critical Mobile Views:**
- Attendance marking (teachers on tablets)
- Student profiles (parents on phones)
- Data tables (horizontal scroll)
- Forms (touch-friendly inputs)
- Navigation (mobile sidebar/hamburger)

**Tasks:**
1. Test all pages at 375px, 768px, 1024px breakpoints
2. Fix touch targets (min 44px)
3. Ensure readable text without zooming
4. Test swipe gestures where applicable

---

### P4.3 Accessibility Audit

**Scope:** WCAG 2.1 AA compliance

**Tasks:**
1. Add ARIA labels to all form controls
2. Fix keyboard navigation in multi-step forms, data tables, modals
3. Focus management verification
4. Color contrast check

---

### P4.4 Performance Optimization

**Tasks:**
1. Add missing `loading.tsx` for slow pages
2. Implement Suspense boundaries for heavy components
3. Verify no N+1 queries (use prisma-optimizer skill)
4. Bundle size audit per route (&lt;100KB target)

---

### P4.5 Security Hardening

**Tasks:**
1. Run `/security-scan` full audit
2. Verify all OWASP Top 10 mitigations
3. Test multi-tenant isolation (no cross-school data leaks)
4. Audit server actions for authorization checks

---

### P4.6 Production Deployment

**Tasks:**
1. Environment variable audit
2. Sentry error monitoring verification
3. Vercel production configuration
4. Database indexes review
5. Create production runbook
