The **Parent Dashboard** (Guardian Dashboard) enables parents to monitor their children's academic progress, attendance, assignments, and school communications.

## Target Users

- **GUARDIAN**: Parents and legal guardians of enrolled students

## Layout Structure

```tsx
<div className="space-y-6">
  {/* Welcome Banner */}
  <WelcomeBanner
    userName={user.name}
    role="Parent"
    subtitle={children.length > 0 ? `Monitoring ${children.length} child(ren)` : "Welcome to the parent portal"}
  />

  {/* Key Metrics Row - 4 columns */}
  <div className="grid gap-4 grid-cols-2 lg:grid-cols-4">
    <MetricCard title="Children" ... />
    <MetricCard title="Attendance" ... />
    <MetricCard title="Pending Tasks" ... />
    <MetricCard title="Announcements" ... />
  </div>

  {/* Attendance Progress Card */}
  <OverallAttendanceCard />

  {/* Quick Actions */}
  <QuickActions actions={...} locale={locale} />

  {/* Main Content Grid - 3 columns */}
  <div className="grid gap-6 lg:grid-cols-3">
    <ChildrenOverviewCard className="lg:col-span-2" />
    <ActivityRings activities={activityData} />
  </div>

  {/* Secondary Content Grid - 2 columns */}
  <div className="grid gap-6 md:grid-cols-2">
    <RecentGradesCard />
    <UpcomingAssignmentsCard />
    <AnnouncementsCard className="md:col-span-2" />
  </div>

  {/* Grade Trend Chart */}
  <ComparisonLineChart ... />

  {/* Progress Cards - 3 columns */}
  <div className="grid gap-6 md:grid-cols-3">
    <ProgressCard title="Attendance Goal" ... />
    <ProgressCard title="Assignments Completed" ... />
    <ProgressCard title="Term Progress" ... />
  </div>
</div>
```

## Welcome Banner

Dynamic child count display:

```tsx
<WelcomeBanner
  userName={user.name || user.email?.split("@")[0]}
  role="Parent"
  subtitle={
    data.children.length > 0
      ? `Monitoring ${data.children.length} ${data.children.length === 1 ? "child" : "children"}`
      : "Welcome to the parent portal"
  }
/>
```

## Key Metrics

| Metric | Icon | Color Logic |
|--------|------|-------------|
| Children | Users | Blue |
| Attendance | Calendar | Green if >=85%, else amber |
| Pending Tasks | FileText | Red if >3, else purple |
| Announcements | Bell | Orange |

## Activity Rings

3 child progress metrics:

```tsx
const activityData = [
  {
    label: "Attendance",
    value: attendanceSummary.percentage,
    color: percentage >= 85 ? "#22c55e" : "#f59e0b",
    current: presentDays,
    target: totalDays,
    unit: "days",
  },
  {
    label: "Grades",
    value: averageGrade,
    color: averageGrade >= 80 ? "#22c55e" : averageGrade >= 60 ? "#3b82f6" : "#ef4444",
    current: Math.round(averageGrade),
    target: 100,
    unit: "%",
  },
  {
    label: "Tasks",
    value: 100 - (notSubmitted.length * 20),
    color: "#8b5cf6",
    current: submitted.length,
    target: total.length,
    unit: "done",
  },
];
```

## Recent Grades

Grade cards for all children:

```tsx
{recentGrades.map((grade) => (
```

## Upcoming Assignments

Assignment list with submission status:

```tsx
{upcomingAssignments.map((assignment) => (
  <div className="flex items-center justify-between p-3 rounded-lg border">
    <div>
      <p className="font-medium">{assignment.title}</p>
      <p className="text-sm text-muted-foreground">
        {assignment.subject} • {assignment.className}
      </p>
    </div>
    <div className="text-right">
      <Badge variant={
        assignment.status === "NOT_SUBMITTED" || isOverdue ? "destructive" :
        assignment.status === "SUBMITTED" ? "default" : "secondary"
      }>
        {assignment.status === "NOT_SUBMITTED" ? "Pending" : assignment.status}
      </Badge>
      <p className="text-xs text-muted-foreground mt-1">
        {isOverdue ? "Overdue" : daysUntil === 0 ? "Due Today" : `${daysUntil} days`}
      </p>
    </div>
  </div>
))}
```

## Data Fetching

Uses `getParentDashboardData()`:

```tsx
const data = await getParentDashboardData();

// Returns:
{
  children: [{ id, name, studentId }],
  attendanceSummary: { presentDays, totalDays, percentage },
  upcomingAssignments: [...],
  recentGrades: [...],
  announcements: [...],
}
```

## Quick Actions

6 actions for GUARDIAN role:

| Action | Icon | Destination |
|--------|------|-------------|
| Children | Users | /children |
| Grades | Award | /grades |
| Attendance | Calendar | /attendance |
| Messages | MessageSquare | /messages |
| Fees | DollarSign | /finance/fees |
| Announce | Bell | /announcements |

## Error Handling

```tsx
try {
  data = await getParentDashboardData();
} catch (error) {
  console.error("[ParentDashboard] Error:", error);
  return <ErrorCard message="Unable to load dashboard" />;
}
```
