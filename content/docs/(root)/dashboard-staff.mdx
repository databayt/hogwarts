The **Staff Dashboard** provides administrative staff with tools to manage daily operations, process requests, track maintenance, and monitor inventory.

## Target Users

- **STAFF**: Administrative and support staff

## Today's Tasks

Task list with status tracking:

```tsx
{
  todayTasks.map((task) => (
    <div className="flex items-center justify-between rounded-lg border p-3">
      <div className="flex items-center gap-3">
        <div
          className={`rounded-full p-2 ${
            task.status === "completed"
              ? "bg-emerald-100"
              : task.status === "in-progress"
                ? "bg-blue-100"
                : "bg-muted"
          }`}
        >
          {task.status === "completed" ? (
            <CircleCheckBig className="h-4 w-4 text-emerald-600" />
          ) : task.status === "in-progress" ? (
            <Clock className="h-4 w-4 text-blue-600" />
          ) : (
            <ListTodo className="text-muted-foreground h-4 w-4" />
          )}
        </div>
        <div>
          <p
            className={`font-medium ${task.status === "completed" ? "text-muted-foreground line-through" : ""}`}
          >
            {task.task}
          </p>
          <p className="text-muted-foreground text-sm">Due: {task.dueTime}</p>
        </div>
      </div>
      <div className="flex items-center gap-2">
        <Badge
          variant={
            task.priority === "high"
              ? "destructive"
              : task.priority === "medium"
                ? "secondary"
                : "outline"
          }
        >
          {task.priority}
        </Badge>
        <Badge
          variant={
            task.status === "completed"
              ? "default"
              : task.status === "in-progress"
                ? "secondary"
                : "outline"
          }
        >
          {task.status.replace("-", " ")}
        </Badge>
      </div>
    </div>
  ))
}
```

Task statuses: pending, in-progress, completed
Priorities: high, medium, low

## Productivity Rings

3 productivity metrics:

```tsx
const productivityRings = [
  {
    label: "Tasks",
    value: (completedToday / totalTasks) * 100,
    color: "#22c55e",
    current: completedToday,
    target: totalTasks,
    unit: "completed",
  },
  {
    label: "Requests",
    value: 100 - pendingRequests.length * 10,
    color: "#3b82f6",
    current: urgentRequests.length,
    target: pendingRequests.length,
    unit: "urgent",
  },
  {
    label: "Queue",
    value: 100 - (overdue / inQueue) * 100,
    color: overdue > 2 ? "#ef4444" : "#f59e0b",
    current: overdue,
    target: 0,
    unit: "overdue",
  },
]
```
