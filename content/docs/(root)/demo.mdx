## Demo Accounts

All accounts use password `1234`.

<div className="flex gap-2 mt-4 mb-4 not-prose">
  <span className="px-2 py-1 text-xs bg-muted rounded">6 roles</span>
  <span className="px-2 py-1 text-xs bg-violet-500/20 text-violet-700 dark:text-violet-400 rounded font-medium">rbac</span>
</div>

- **admin@databayt.org** → Full access, manage everything <span className="ml-2 px-1.5 py-0.5 text-[10px] bg-red-500/20 text-red-700 dark:text-red-400 rounded font-medium">admin</span>
- **teacher@databayt.org** → Assigned classes, attendance, grades <span className="ml-2 px-1.5 py-0.5 text-[10px] bg-blue-500/20 text-blue-700 dark:text-blue-400 rounded font-medium">teacher</span>
- **student@databayt.org** → Personal schedule, grades, assignments <span className="ml-2 px-1.5 py-0.5 text-[10px] bg-green-500/20 text-green-700 dark:text-green-400 rounded font-medium">student</span>
- **parent@databayt.org** → View linked children data (read-only) <span className="ml-2 px-1.5 py-0.5 text-[10px] bg-yellow-500/20 text-yellow-700 dark:text-yellow-400 rounded font-medium">guardian</span>
- **staff@databayt.org** → Limited admin on assigned entities <span className="ml-2 px-1.5 py-0.5 text-[10px] bg-cyan-500/20 text-cyan-700 dark:text-cyan-400 rounded font-medium">staff</span>
- **accountant@databayt.org** → Billing and invoices only <span className="ml-2 px-1.5 py-0.5 text-[10px] bg-amber-500/20 text-amber-700 dark:text-amber-400 rounded font-medium">finance</span>
