هذه مركز تنفيذ المبيعات. افتحها لترى ما أُنجز، وما هو التالي، وأين تتعلّم كل خطوة. **الحالة الحالية: تم توقيع تجربة واحدة من أصل 10 في MENA-10** — [مدارس الملك فهد](https://kingfahad.databayt.org)، العضو المؤسِّس في المجموعة التجريبية المجانية. كل مهمة أدناه تربط بالمستند الذي يشرح كيفية إنجازها. تواريخ المراحل وبواباتها مملوكة لـ [خطة الانتشار](/ar/docs/gtm).

## لمحة الحالة

المراحل الرئيسية لعام 2026، مصدرها [خطة الانتشار](/ar/docs/gtm) المرحلة 1. حدِّث عمود الحالة مع إنجاز كل مرحلة.

| المرحلة                         | الهدف      | المسؤول       | الحالة |
| ------------------------------- | ---------- | ------------- | ------ |
| تحويل مدارس الملك فهد إلى مدفوع | 2026-06-15 | سيدون، عبدالـ | ☐      |
| 3 تجارب سودانية تعمل            | 2026-07-31 | علي           | ☐      |
| توقيع 10 تجارب MENA-10          | 2026-09-30 | علي، عبدالـ   | 1 / 10 |
| أول تحويل مدفوع، ≥ $99 شهرياً   | 2026-10-31 | عبدالـ        | ☐      |
| 3 دراسات حالة منشورة            | 2026-11-30 | سامية         | ☐      |

راجع [خطة الانتشار](/ar/docs/gtm) للخطة الكاملة بمراحلها الثلاث، ومنطق بوابات المراحل، وخريطة المسؤولين.

## خطة التنفيذ

اعمل من الأعلى إلى الأسفل. كل مهمة تربط بالمستند الذي يشرح كيفية إنجازها.

### 0. الأساس والمحتوى الإبداعي

ابدأ بها أولاً — فهي تفتح كل محادثة لاحقة.

- [ ] انشر دراسة حالة مدارس الملك فهد بأرقام حقيقية ← [دراسة الحالة](/ar/docs/case-study)
- [ ] سجِّل فيديو العرض التوضيحي (دقيقتان) بتعليق صوتي عربي ← [التسويق](/ar/docs/marketing)
- [ ] التقط حزمة لقطات الشاشة (فاتح + داكن، عربي + إنجليزي) ← [التسويق](/ar/docs/marketing)
- [ ] ثبِّت الشعارات النصية العربية والإنجليزية ← [التسويق](/ar/docs/marketing)
- [ ] أنتج فيديو حزمة الترحيب للتجارب ← [البرنامج التجريبي](/ar/docs/pilot)

### 1. بناء خط الأنابيب (التنقيب)

- [ ] جمِّع قائمة مدارس مُسمّاة عبر نطاق MENA-10 ← [العملاء المحتملون](/ar/docs/leads)
- [ ] أهِّل كل مدرسة عبر نص الاكتشاف ذي الأسئلة السبعة ← [العملاء المحتملون](/ar/docs/leads)

### 2. التواصل (بارد + دافئ)

- [ ] نفِّذ تواصلاً من 5 لمسات عبر البريد وWhatsApp وLinkedIn ← [التواصل](/ar/docs/outreach)
- [ ] التزم بإيقاع المتابعة لليوم 1 / 3 / 7 / 14 / 21 ← [العملاء المحتملون](/ar/docs/leads)
- [ ] اطلب من الشبكة تعريفات دافئة (أحمد بها، سيدون، MSAR) ← [العملاء المحتملون](/ar/docs/leads)

### 3. العرض التجاري والعرض التوضيحي

- [ ] قدِّم عرضاً حياً مدته 30 دقيقة (عربي / RTL، وجهات نظر حسب الدور) ← [العرض التوضيحي](/ar/docs/demo)
- [ ] اكتب عرض التجربة المجانية ← [العرض التجاري](/ar/docs/proposal)
- [ ] أرسله ونفِّذ متابعة العرض على مدى 21 يوماً ← [العرض التجاري](/ar/docs/proposal)

### 4. تهيئة التجربة (اليوم 0 ← 180)

- [ ] اليوم 0 — جهِّز النطاق الفرعي وحمِّل بذرة المنهج ← [البرنامج التجريبي](/ar/docs/pilot)
- [ ] اليوم 1 — مكالمة التهيئة مع حزمة الترحيب العربية ← [البرنامج التجريبي](/ar/docs/pilot)
- [ ] اليوم 7 / 14 / 30 / 60 — فحوص الصحة، فرز الطلبات، NPS، المراجعة ← [البرنامج التجريبي](/ar/docs/pilot)
- [ ] اليوم 90 — مسوّدة دراسة الحالة ← [دراسة الحالة](/ar/docs/case-study)

### 5. التحويل إلى مدفوع

- [ ] اليوم 180 — أرسل عرض التحويل (خصم 30٪ على PRO) ← [البرنامج التجريبي](/ar/docs/pilot)
- [ ] طبِّق منطق بوابات المراحل قبل توسيع الإنفاق ← [خطة الانتشار](/ar/docs/gtm)
- [ ] سعِّر الصفقة (المستويات، الخصم السنوي، شروط متعددة المدارس) ← [نموذج العمل](/ar/docs/business-model)

## الإيقاع الأسبوعي

أهداف نشاط متكرّرة من [العملاء المحتملون](/ar/docs/leads). تُعاد كل يوم اثنين؛ يرفع القائد إشارة خطر إذا بقي أي هدف دون المستوى لأسبوعين.

- [ ] 3 عملاء محتملون دافئون جدد
- [ ] 10 إجراءات تواصل
- [ ] مكالمتا اكتشاف
- [ ] عرض واحد مُرسَل (شهرياً)
- [ ] تجربتان موقَّعتان (شهرياً)

## مرجع

الأساسيات المختصرة. التفاصيل الكاملة في المستندات المرتبطة.

**لمن نبيع** — مدارس K-12 في الشرق الأوسط وأفريقيا، عربية أولاً. أولوية الجغرافيا: السودان (نقطة الانطلاق) ← السعودية (أكبر سوق) ← مصر ← باقي الخليج. المشترون هم المديرون ومالكو المدارس ووزارات التعليم وشبكات المدارس. الحجم الكامل في [نموذج العمل](/ar/docs/business-model).

**التسعير**

| المستوى      | السعر          | الأنسب لـ                           |
| ------------ | -------------- | ----------------------------------- |
| مجاني (FREE) | $0             | ≤ 100 طالب، التقييم                 |
| PRO          | $1.50/طالب/شهر | المدارس النامية، علامة تجارية مخصصة |
| ENTERPRISE   | $1.00/طالب/شهر | الشبكات، العقود الحكومية            |

مدرسة 500 طالب على PRO تكلِّف $750/شهر؛ مقارنة المستويات واقتصاديات الوحدة في [نموذج العمل](/ar/docs/business-model).

**أبرز الاعتراضات** (النصوص الكاملة في [العملاء المحتملون](/ar/docs/leads) و[المنافسون](/ar/docs/competitors))

- _"لدينا نظام بالفعل"_ — Hogwarts مبنية لمنطقتكم منذ اليوم الأول: عربية أولاً، عزل متعدد المستأجرين، قنوات دفع إقليمية.
- _"إنها باهظة"_ — المدرسة النموذجية توفّر ~40 ساعة إدارية/شهر؛ عند $1.50/طالب تسترد كلفتها خلال 2–3 أشهر.
- _"نحتاج إشراك الوزارة / المجلس"_ — لهذا توجد التجربة المجانية لستة أشهر: قِس، وثِّق، ثم قرِّر.
