## الحزمة الأساسية

| التقنية          | الإصدار         | الغرض                                                 |
| ---------------- | --------------- | ----------------------------------------------------- |
| **Next.js**      | 16.2.x          | App Router، Turbopack، Server Components، streaming   |
| **React**        | 19.2.x          | مكتبة UI مع concurrent features وServer Components    |
| **TypeScript**   | 5.x             | strict mode، أمان نوع عبر الحزمة                      |
| **Prisma**       | 6.19.x          | ORM مع 302 نموذجاً، 149 enum، 73 ملف schema           |
| **PostgreSQL**   | Neon Serverless | قاعدة بيانات متعدّدة المستأجرين مع connection pooling |
| **NextAuth**     | v5 beta.30      | OAuth (Google, Facebook)، credentials، جلسات JWT      |
| **Tailwind CSS** | 4               | utility-first مع تنسيق ألوان OKLCH                    |
| **shadcn/ui**    | الأحدث          | Radix UI primitives، مكونات متاحة (accessible)        |

## Runtime والنشر

| التقنية           | الغرض                                            |
