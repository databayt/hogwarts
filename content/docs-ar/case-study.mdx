دراسة الحالة هي دليل. هي القطعة التي تُحوِّل التجربة الاسترشادية التالية، المدرسة المدفوعة التالية، والمساهم التالي. نَهدف إلى نشر دراسة حالة واحدة لكل تجربة استرشادية — ثلاث بحلول Q3 2026، ست بحلول Q1 2027. الأولى قيد التقدُّم مع [مدارس الملك فهد](https://kingfahad.databayt.org). لاستراتيجية التجربة الاسترشادية انظر [التجربة الاسترشادية](/docs/pilot)؛ للتوزيع التسويقي انظر [التسويق](/docs/marketing).

## لماذا تَهم دراسات الحالة

| للجمهور                                | ما تَفعله دراسة الحالة                                  |
| -------------------------------------- | ------------------------------------------------------- |
| التجربة الاسترشادية المُحتمَلة التالية | "مدرسة مثل مدرستي تَفعل هذا — أنا لست الأول."           |
| المستثمرون / الحاضنات                  | تُؤكِّد أن المنتج قيد الاستخدام، لا في الشرائح          |
| المساهمون                              | تُظهر أن المنتج يُؤثِّر في طلاب ناطقين بالعربية حقيقيين |
| الصحافة                                | تُوفِّر قصة، اسمًا، وجهًا، اقتباسًا                     |
| المشتريات الحكومية                     | تَخفض مخاطر القرار؛ تُوفِّر مكالمة مرجع                 |
| الفريق الداخلي                         | تُذكِّرنا لماذا نَقوم بالعمل                            |

فقرة واحدة من مدير حقيقي تَهزم عشر فقرات منَّا.

## تشريح دراسة حالة هَوْغوارتس

كل دراسة حالة تَستخدم نفس الهيكل الخماسي. هدف الطول: 600-900 كلمة. تَبقى في صفحة واحدة عند الطباعة.

| #   | القسم                | الغرض                                                               | الطول    |
| --- | -------------------- | ------------------------------------------------------------------- | -------- |
| 1   | عنوان + بيانات وصفية | المدرسة، البلد، الحجم، تاريخ بدء التجربة — يُوجِّه القارئ فورًا     | 1-2 سطر  |
| 2   | المشكلة              | كيف بَدا يوم المدرسة قبل هَوْغوارتس (مَلموس، بِساعات/نسب)           | 150 كلمة |
| 3   | ما فعلناه            | التجربة الاسترشادية — التهيئة، التدريب، التخصيصات، الجدول الزمني    | 200 كلمة |
| 4   | النتائج              | ساعات مُوفَّرة، أهالي وُصِل إليهم، أخطاء مُخفَّضة — مُقدَّرة كميًّا | 200 كلمة |
| 5   | الاقتباس             | اقتباس من المدير أو إداري                                           | 1-3 جمل  |

## خطة النشر

1. **النشر على الموقع** — `ed.databayt.org/case-studies/<slug>` صفحة عربي + إنجليزي.
2. **تَجزئة** — منشور LinkedIn + GitHub README + رسالة WhatsApp تَستشهد بالأرقام الرئيسية.
3. **توزيع** — مع كل اقتراح [التجربة الاسترشادية](/docs/pilot)، تَشمل أحدث دراسة حالة كمُلحق.
4. **اعتراف الراعي** — إذا كانت التجربة الاسترشادية مدعومة من راعٍ (CSR، NGO، حاضنة)، اِنسب الفضل في الرأس.
5. **ائتمان المساهم** — اِنسب الفضل لجميع المساهمين عبر [الاقتصاد المُشترَك](/docs/shared-economy).
