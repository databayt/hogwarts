## إعداد بيئة التطوير المحلية

**01. تثبيت VS Code**

قم بالتحميل من [code.visualstudio.com](https://code.visualstudio.com/)

---

**02. تثبيت Node.js**

قم بالتحميل من [nodejs.org](https://nodejs.org/) (إصدار LTS)

---

**03. تثبيت Git**

قم بالتحميل من [git-scm.com](https://git-scm.com/downloads)

---

**04. فتح VS Code**

قم بتشغيل VS Code، ثم افتح الطرفية: View → Terminal

---

**05. تثبيت pnpm**

```text
npm install -g pnpm
```

---

**06. استنساخ المستودع**

```bash
git clone https://github.com/databayt/hogwarts.git
```

---

**07. فتح المشروع في VS Code**

File → Open Folder → اختر مجلد `hogwarts`

---

**08. فتح الطرفية في VS Code**

View → Terminal أو `` Ctrl + ` ``

---

**09. تثبيت التبعيات**

```bash
pnpm install
```

---

**10. إنشاء ملف البيئة**

أنشئ `.env` في المجلد الجذر والصق المحتوى من مجموعة واتساب

---

**11. إنشاء عميل Prisma**

```bash
pnpm prisma generate
```

---

**12. إعداد قاعدة البيانات**

```bash
pnpm prisma migrate dev
```

---

**13. بدء خادم التطوير**

```bash
pnpm dev
```

---

**14. فتح المتصفح**

اذهب إلى http://localhost:3000

_للإعداد التفصيلي، انظر [إعداد بيئة التطوير المحلية](/ar/docs/localhost)._

## أول مشكلة جيدة

أضف اسمك إلى صفحة المدونة كأول مساهمة لك.

**01. إنشاء فرع جديد**

```bash
git checkout -b feat/your-name
```

---

**02. فتح الملف**

افتح `src/components/marketing/blog/content.tsx` في VS Code

---

**03. إيجاد نص Blog**

ابحث عن هذا الكود:

```tsx
<div>Blog</div>
```

---

**04. أضف اسمك أسفل Blog**

غيّره إلى:

```tsx
<div>
  Blog
  <p>اسمك</p>
</div>
```

---

**05. حفظ الملف**

اضغط `Ctrl + S`

---

**06. إضافة التغييرات للمرحلة**

```bash
git add .
```

---

**07. تثبيت التغييرات**

```bash
git commit -m "feat: add اسمك to blog"
```

---

**08. الدفع إلى GitHub**

```bash
git push origin feat/your-name
```

**هذا كل شيء! مرحباً بك في هوجوورتس.**
