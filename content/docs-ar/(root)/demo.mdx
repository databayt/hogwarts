## جولة سريعة

الأماكن الأساسية للاستكشاف في 5 دقائق.

<div className="not-prose mt-4 mb-4 flex gap-2">
  <span className="bg-muted rounded px-2 py-1 text-xs">5 خطوات</span>
  <span className="rounded bg-green-500/20 px-2 py-1 text-xs font-medium text-green-700 dark:text-green-400">
    موصى به
  </span>
</div>

- **الإعدادات** ← غيّر الدور لرؤية لوحات تحكم مختلفة <span className="ml-2 px-1.5 py-0.5 text-[10px] bg-orange-500/20 text-orange-700 dark:text-orange-400 rounded font-medium">جرّب أولاً</span>
- **المظهر** ← بدّل الوضع الداكن/الفاتح واللغة <span className="ml-2 px-1.5 py-0.5 text-[10px] bg-blue-500/20 text-blue-700 dark:text-blue-400 rounded font-medium">i18n</span>
- **الطلاب** ← تصفح القائمة، أضف طالب، عرض تبويبات الملف <span className="ml-2 px-1.5 py-0.5 text-[10px] bg-emerald-500/20 text-emerald-700 dark:text-emerald-400 rounded font-medium">crud</span>
- **الجدول** ← منشئ الجدول بالسحب والإفلات <span className="ml-2 px-1.5 py-0.5 text-[10px] bg-purple-500/20 text-purple-700 dark:text-purple-400 rounded font-medium">تفاعلي</span>
- **الحضور** ← سجّل الحضور اليومي للفصل <span className="ml-2 px-1.5 py-0.5 text-[10px] bg-pink-500/20 text-pink-700 dark:text-pink-400 rounded font-medium">تتبع</span>
