طريق سريع من الصفر إلى التشغيل. للتفاصيل الأعمق راجع [Localhost](/ar/docs/localhost).

## المتطلبات الأولية

- [Node.js](https://nodejs.org/) (LTS) و[pnpm](https://pnpm.io/installation): `npm install -g pnpm`
- [Git](https://git-scm.com/downloads)
- محرر شفرات — VS Code يُنصح به

## التشغيل المحلي

```bash
git clone https://github.com/databayt/hogwarts.git
cd hogwarts
pnpm install
pnpm prisma generate
pnpm prisma migrate dev
pnpm dev
```

افتح [http://localhost:3000](http://localhost:3000). سجّل الدخول بـ `admin@databayt.org` / `1234`.

ملف `.env` يُشارَك في مجموعة WhatsApp — اطلب صلاحية الوصول إن لم تكن لديك بعد.

## أول PR لك

أضف اسمك إلى صفحة المدونة في ثلاث commits أو أقل.

<Steps>

<Step>إنشاء فرع وتعديل</Step>

```bash
git checkout -b docs/your-name
```

افتح `src/components/marketing/blog/content.tsx` وأضف سطراً تحت عنوان `Blog`:

```tsx
<div>
  Blog
  <p>Your Name</p>
</div>
```

<Step>Commit ودفع</Step>

```bash
git add src/components/marketing/blog/content.tsx
git commit -m "feat(marketing): add your-name to blog"
git push origin HEAD
```

<Step>افتح PR</Step>

العنوان: قصير ومحدّد. الوصف: ما الذي تغيّر ولماذا. أرفق لقطة شاشة إن غُيِّرت الواجهة. أهلاً بك في الفريق.

</Steps>
