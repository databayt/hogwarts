اشتراك SaaS متعدّد المستويات بتسعير لكل طالب. مصمَّم ليتوسّع من مدرسة خاصة من 50 طالباً إلى شبكة وطنية من المدارس الحكومية. يعمل في الإنتاج على [ed.databayt.org](https://ed.databayt.org)؛ ويُجرَّب مع [مدارس الملك فهد](https://kingfahad.databayt.org) — أوّل عميل مدفوع لنا. للعملية التجارية راجع [المبيعات](/en/docs/sales).

## مستويات الاشتراك

| المستوى        | السعر          | الأنسب لـ                | الحدود                                                                                                            |
| -------------- | -------------- | ------------------------ | ----------------------------------------------------------------------------------------------------------------- |
| **FREE**       | $0             | المدارس الصغيرة، التقييم | 100 طالب، 1 GB تخزين، دعم مجتمعي، الميزات الأساسية                                                                |
| **PRO**        | $1.50/طالب/شهر | المدارس النامية          | 10 GB تخزين، دعم ذو أولوية، علامة تجارية مخصّصة، API بمعدل 1K req/hr، تحليلات متقدّمة، إشعارات دفع لأولياء الأمور |
| **ENTERPRISE** | $1.00/طالب/شهر | الشبكات، العقود الحكومية | تخزين وAPI غير محدودَين، مدير حساب مخصّص، SLA بنسبة 99.9%، علامة بيضاء، SSO عبر (SAML/OAuth)، تكاملات مخصّصة      |

كل المستويات المدفوعة تشمل **تجربة مجانية لمدة 14 يوماً** بوصول كامل للميزات؛ بدون بطاقة ائتمان.

## خيارات الفوترة

| الخيار          | الخصم   | الفوترة                       |
| --------------- | ------- | ----------------------------- |
| شهرية           | —       | لكل طالب، محسوبة شهرياً       |
| سنوية           | خصم 15% | التزام سنوي مدفوع مقدّماً     |
| متعدّدة الأعوام | مخصّص   | يُتفاوض عليها للعقود الحكومية |

**طرق الدفع**: Stripe Subscriptions (الأساسية)، التحويل البنكي (المؤسسات / الحكومات)، البوّابات الإقليمية (bankak، tap، النقد عبر الجوّال) لمنطقة MEA.

## مصادر الإيراد

| المصدر                | الحصة | الوصف                                          |
| --------------------- | ----- | ---------------------------------------------- |
| اشتراكات لكل طالب     | ~85%  | إيراد متكرّر من الخطط الشهرية / السنوية        |
| عقود المؤسسات         | ~10%  | اتفاقيات متعدّدة المدارس + حكومية بتسعير مخصّص |
| تجاوزات API           | صغير  | فوترة قائمة على الاستخدام تتجاوز حدود المستوى  |
| ترخيص العلامة البيضاء | صغير  | رسوم نطاق مخصّص + إعادة علامة تجارية كاملة     |
| الخدمات الاحترافية    | صغير  | التطبيق، الترحيل، التدريب                      |
| السوق (مستقبلاً)      | —     | عمولة على التكاملات من أطراف ثالثة             |

## الفرصة السوقية (MEA)

| الإقليم                 | حجم 2024 | المتوقَّع         | CAGR   |
| ----------------------- | -------- | ----------------- | ------ |
| EdTech في MEA           | $14.4B   | $33.0B بحلول 2030 | ~14.6% |
| EdTech الشرق الأوسط     | $11.3B   | $27.0B بحلول 2033 | ~9.1%  |
| EdTech دول الخليج (GCC) | $3.02B   | $4.47B بحلول 2030 | ~6.74% |

يمتلك قطاع K-12 نسبة **44%** من سوق EdTech — وهو القطاع الأكبر في توليد الإيراد. يتجاوز إجمالي TAM لـK-12 في MEA حاجز **$6B بحلول 2030**.

## بنية التكلفة

| الفئة         | الخدمة                                                                          |
| ------------- | ------------------------------------------------------------------------------- |
| الاستضافة     | Vercel (serverless)، Neon (PostgreSQL serverless)                               |
| التخزين / CDN | AWS S3 + CloudFront (`cdn.databayt.org`)                                        |
| البريد / SMS  | Resend، Twilio                                                                  |
| المدفوعات     | Stripe (2.9% + $0.30/معاملة)، البوّابات الإقليمية                               |
| التطوير       | الفريق الأساسي + مساهمو المجتمع عبر [الاقتصاد المشترك](/ar/docs/shared-economy) |
| الامتثال      | تدقيق أمني سنوي، شهادات                                                         |

### الهوامش المستهدفة

- **هامش الربح الإجمالي**: 80%+ (كفاءة سحابية أصيلة)
- **هامش التشغيل**: 20%+ على نطاق واسع
- **نسبة LTV:CAC**: 10:1+

## السوق والشراكات

سوق تكامل مستقبلي (تقاسم 70/30 بين الشريك والمنصة) وشبكة مُعيدي بيع ضمن خارطة الطريق؛ غير مفعَّلة اليوم.

## الحصص وتقاسم الإيراد

كل الموظفين يحصلون على حصص (استحقاق 4 سنوات، cliff سنة واحدة). مساهمو المصدر المفتوح يكسبون عبر إطار [الاقتصاد المشترك](/ar/docs/shared-economy).
