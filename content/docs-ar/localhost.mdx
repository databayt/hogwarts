## المتطلبات الأولية

- [Node.js](https://nodejs.org/) v22+ (يُنصح بـ LTS)

## 3. تثبيت التبعيات

افتح Terminal المدمج (`Ctrl/Cmd + ~`):

```bash
pnpm install
```

## 4. إعداد البيئة

أنشئ ملف `.env` في جذر المشروع والصق محتواه من مجموعة WhatsApp. لا يوجد سوى **`.env` واحد** — لا `.env.local` ولا `.env.development` ولا غيرها.

## 5. توليد Prisma client وتشغيل الـ migrations

```bash
pnpm prisma generate
pnpm prisma migrate dev
```

## 6. تشغيل مخدّم التطوير

```bash
pnpm dev
```

افتح [http://localhost:3000](http://localhost:3000). سجّل الدخول بـ `admin@databayt.org` / `1234`.
