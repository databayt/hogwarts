جرّب المنصة مباشرة. افتح [demo.databayt.org/ar/dashboard](https://demo.databayt.org/ar/dashboard) وسجّل الدخول بـ `admin@databayt.org` / `1234`.

## جولة سريعة

خمسة مواقع لاستكشافها في خمس دقائق.

1. **الإعدادات** — بدّل الدور لرؤية لوحات تحكم مختلفة.
2. **المظهر** — بدّل بين الوضع الداكن والفاتح، وبين العربية والإنجليزية.
3. **الطلاب** — تصفّح القائمة، أضف طالباً، اطّلع على تبويبات الملف الشخصي.
4. **الجدول الدراسي** — منشئ الجدول بالسحب والإفلات.
5. **الحضور** — تسجيل الحضور اليومي للفصل.

## حسابات العرض

كل الحسابات تستخدم كلمة المرور `1234`.

| البريد الإلكتروني         | الدور      | الصلاحيات                                |
| ------------------------- | ---------- | ---------------------------------------- |
| `admin@databayt.org`      | ADMIN      | وصول كامل، إدارة كل شيء                  |
| `teacher@databayt.org`    | TEACHER    | الفصول المكلَّف بها، الحضور، الدرجات     |
| `student@databayt.org`    | STUDENT    | الجدول الشخصي، الدرجات، الواجبات         |
| `parent@databayt.org`     | GUARDIAN   | عرض بيانات الأبناء المرتبطين (قراءة فقط) |
| `staff@databayt.org`      | STAFF      | إدارة محدودة على الكيانات المخصَّصة      |
| `accountant@databayt.org` | ACCOUNTANT | الفوترة والفواتير فقط                    |

## القيود

نسخة العرض تُعاد ضبطها يومياً وتعمل في وضع sandbox:

- إشعارات البريد معطّلة.
- رفع الملفات محدود بـ 1 ميغابايت.
- لا يمكن إنشاء حسابات مستخدمين جديدة.
- إعدادات المدرسة مقفلة.
