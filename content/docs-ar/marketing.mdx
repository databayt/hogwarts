التسويق لـ هَوْغوارتس قائم على المحتوى، عربي-أولًا، مُحرَّك بالتجربة الاسترشادية. لا نَشتري إعلانات في 2026؛ نَبني الدليل. كل قطعة محتوى إما تُعلِّم مديرَ مدرسة شيئًا مفيدًا، أو تُظهر المنتج، أو تُضخِّم دراسة حالة تجربة استرشادية. لحيث يَلتقي التسويق بالمبيعات انظر [GTM](/docs/gtm)؛ لخطافات تجنيد التجربة الاسترشادية انظر [التجربة الاسترشادية](/docs/pilot).

## التموقُع

> نظام تشغيل مدرسي عربي-أولًا، مفتوح المصدر للشرق الأوسط — مبني للمدارس التي تَدخل الخدمة، لا للمدارس التي تُحدِّث أنظمة قديمة.

| الركيزة            | ما تَعنيه                                                        | الدليل                                                  |
| ------------------ | ---------------------------------------------------------------- | ------------------------------------------------------- |
| عربي-أولًا         | RTL أصلي، كل شاشة، كل دور، كل بريد                               | وثائق [التعريب](/docs/internationalization)             |
| مفتوح المصدر       | SSPL، خريطة طريق عامة، فشل عام، مساهمون كمالكين                  | [الاقتصاد المُشترَك](/docs/shared-economy)              |
| مَبني للشرق الأوسط | تجربة استرشادية في السودان أولًا، فريق مؤسِّس مُجذَّر في المنطقة | [تجربة مدارس الملك فهد](https://kingfahad.databayt.org) |
| مدارس تَدخل الخدمة | مبني لسياقات K-12 ما بعد الصراع، الأسواق الناشئة، العربية        | سياق السودان في [الحاضنات](/docs/incubators)            |

لسنا الأرخص، ولسنا الأكبر، ولسنا الأكثر تَلميعًا. نحن الأكثر تَوافُقًا مع كيفية عمل المدارس الخاصة في الشرق الأوسط فعلًا.

## شرائح الجمهور

| الشريحة               | من هم                                | القناة                           | ما يَهتمُّون به                            |
| --------------------- | ------------------------------------ | -------------------------------- | ------------------------------------------ |
| ملاك K-12 الخاص       | 1-3 مدارس، إيراد سنوي 50K-500K       | LinkedIn (عربي)، WhatsApp        | الساعات المُوفَّرة، رضا ولي الأمر، التكلفة |
| إداريو المدارس        | المُشغِّلون اليوميون                 | LinkedIn (عربي)، إحالة أقران     | الأعمال الورقية، الحضور، مكالمات الأهل     |
| المشتريات الحكومية    | صانعو قرار وزارة / منطقة             | RFP + تقديمات                    | الامتثال، متعدد المدارس، عقود مُخصَّصة     |
| شركاء التعليم التقني  | مُزوِّدو محتوى، مراكز تدريب          | تواصُل مباشر، مؤتمرات            | التكامل، تقاسم الإيراد                     |
| مساهمو المصدر المفتوح | مطوِّرون في الشرق الأوسط، طلاب، هواة | GitHub، Twitter، Discord         | المنصة، مشاكل لحلها، الاعتراف              |
| جمعيات أولياء الأمور  | أصوات مجتمعية مؤثِّرة                | علاقات مدارس التجارب الاسترشادية | جودة التواصل، الشفافية                     |

## القنوات الأساسية (2026)

| القناة                             | الجمهور المستهدف             | إيقاع                   |
| ---------------------------------- | ---------------------------- | ----------------------- |
| LinkedIn (عربي)                    | ملاك المدارس + الإداريون     | 3 منشورات / أسبوع       |
| GitHub + الوثائق                   | المساهمون + المطوِّرون       | عند الميزة + إصدار شهري |
| WhatsApp                           | مديرو التجارب + علاقات إحالة | حسب الحاجة              |
| المؤتمرات (GESS، BETT، Misk، LEAP) | المشتريات + الشراكات         | 4 / سنة                 |
| محتوى دراسة حالة                   | جميع الشرائح                 | عند تحويل + موسمي       |
| ندوات عبر الإنترنت                 | الإداريون                    | 1 / شهر                 |

## تقويم المحتوى (نموذج)

- ملف ميزة شهري
- دراسة حالة ربعية
- صدارة فكرية أسبوعية
- محتوى ولي أمر + معلِّم متبادل في القناة

## أهداف SEO

كلمات مفتاحية عربية K-12 مُتخصِّصة: "نظام إدارة مدرسي عربي"، "برنامج مدرسة سحابي"، "نظام حضور آلي"، "مدفوع لكل طالب"، "مفتوح المصدر للمدارس".
