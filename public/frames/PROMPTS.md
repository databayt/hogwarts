# School Onboarding Video Frame Prompts

## Single Prompt (upload all 13 frames as reference)

```
Generate 13 frames for a school onboarding animation in the EXACT same isometric 3D render style as the reference images. Same soft studio lighting, clean white background, warm pastel palette, geometric minimalist rendering, same 770x674 dimensions. No text, no people.

The reference images show a home being built across 3 stages. Recreate the same progressive narrative but for a SCHOOL:

STAGE 1 — "Tell us about your school" (4 frames, isometric interior cutaway like reference frames 1-4):
Frame 1: Empty floor plan, partial walls just rising, concrete outline only
Frame 2: Walls formed, two classrooms and hallway visible, bare floors, window openings
Frame 3: Basic wooden desks in rows, whiteboard on wall, empty bookshelves upstairs, staircase installed, grid windows
Frame 4: Two-story cutaway complete — classroom with desks and whiteboard downstairs, library with shelves and reading table upstairs, tiled floors, everything clean but minimal and unfurnished

STAGE 2 — "Make it stand out" (4 frames, same interior cutaway, adding warmth and personality like reference frames 5-8):
Frame 5: Books and pencil holders appear on desks, world map poster, globe on library table, shelves half-filled
Frame 6: Alphabet banner across classroom wall, student art displayed, teacher's desk with plant, library reading corner with blue armchair, colorful rug, more books
Frame 7: Cubbies with backpacks, coat hooks, trophy on shelf, curtains on windows, educational posters everywhere, warm golden light
Frame 8: Fully alive — desks full of supplies, aquarium on shelf, walls covered in student art, library packed with colorful books, reading cushions, floor lamp, every detail says this school is loved

STAGE 3 — "Finish up and publish" (5 frames, zoom out to exterior like reference frames 9-13):
Frame 9: Basic school exterior, two-story, clean white/gray facade, flat modern roof, large windows, simple front entrance, just a walkway
Frame 10: Flagpole with flag, awning over entrance, school sign, young trees planted, bench near front
Frame 11: Trimmed hedges, flower beds, playground with swings on one side, bike rack, stone pathway, green lawn
Frame 12: Clock tower element on roof, skylights, sports court visible, bikes parked, warm classrooms glowing through windows
Frame 13: Complete and ready — lush mature trees, colorful playground, manicured gardens, golden hour lighting, warm interior visible through every window, welcome mat at the door, the school radiates warmth and community

Maintain perfect style consistency across all 13 frames. The progression should feel like one continuous animation: bare structure → decorated interior → finished exterior.
```

## Tips for Best Results

1. **Upload the original frame** as reference/style image alongside the prompt
2. **Aspect ratio**: Use 770x674 (or close to it) to match originals
3. **If the style drifts**, prepend: "In the exact same isometric 3D render style as the reference image,"
4. **Consistency across frames**: Generate all frames for one stage in one session if possible
5. **After generation**: Use the same frames directory structure:
   - `school-stage1-frame-01.png` through `school-stage1-frame-04.png`
   - `school-stage2-frame-01.png` through `school-stage2-frame-04.png`
   - `school-stage3-frame-01.png` through `school-stage3-frame-05.png`
